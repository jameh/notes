More NP Complete Problems¶

Hamiltonian Cycle¶

Find a cycle going through every vertex exactly once.

Hamiltonian Path¶

Find a path going through every vertex exactly once.

Maze Problem¶

Find a path from \(v_1\) to \(v_2\) going through no vertex more than once, collect as much gold as possible.

Given an instance of Hamiltonian Path,

If a Maze-Problem algorithm existed, then applying it to \(v_1\) and \(v_2\) would answer “max-value=n” \(\iff\) there exists a Hamiltonian Path from \(v_1\) to \(v_2\).

Note

Each Hamiltonian Path is equivalent to multiple Maze Problems.

This is called a “one-to-many” type reduction.

Subset Sum (again)¶

Given \(S=\{\text{set of integers}\}\), and \(k\) an integer, Does \(S\) have a subset that sums to \(k\)?

This problem is NP-Complete.

Some sets of integers are easy, some are hard. e.g. \(\{1,2,4,...,2^n\}.\)

The Brute-Force Ignorant algorithm for subset sum is: try every subset of \(S\). (there are \(2^n\) of them), so \(O(2^n)\) steps.

We can actually do a little better :)

Given 2 sets of size \(m\), can we find two elements, one from each set, that sum to a given target value?

Brute-Force Algorithm for this is \(O(m^2)\)

Horowitz & Sahni algorithm:

  • Sort the first set (\(S_1\)) into ascending order, and the second set (\(S_2\)) into descending order.
  • \(S_1: x_1 \leq x_2 \leq ... \leq x_m\)
  • \(S_1: y_1 \leq y_2 \leq ... \leq y_m\)
  • Consider \(x_1+y_1\). If \(x_1+y_1=k\), then we’re done.
  • else if \(x_1+y_1 > k\), then we know \(x_i+y_1>k\) for all \(i \geq 1\).
  • else (\(x_1+y_1 < k\)), then we know \(x_1+y_i<k\) for all \(i \geq 1\).

Every iteration eliminates a pair of numbers. Sorting takes \(O(m\log m)\) and the rest takes \(O(m)\). That’s better than \(O(m^2)\) :)

Horowitz & Sahni Algorithm for Subset Sum¶

Given \(S=\{s_1,...s_n\}\), \(k\)

split \(S\) into \(P\) and \(Q\), equal size.

Find all subset sums in \(P\), call this set \(X_P\)

And find all subset sums in \(Q\), call this set \(X_Q\).

Example

\[P=\{1,2,3\}\]\[X_P = \{0,1,2,3,4,5,6\}\]

Sort \(X_P\) and \(X_Q\).

\[O(2^{n/2}\log 2^{n/2})\]\[= O(n*2^{n/2})\]
