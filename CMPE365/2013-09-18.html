More Decision Problems¶

\(P\): decision problems that can be solved in polynomial time

\(NP\): decision problems that can be verified in polynomial time

  • only verify yes answers
\[P \subseteq NP\]
  • for example, if \(p \in P\), then \(p\) has a solution algorithm in polynomial time - so run the solution algorithm again for verification - therefore it can be verified in polynomial time and \(p \in NP\)

Is it a strict inclusion? i.e. does there exist a \(NP\) problem that is not in \(P\)?

Problem Reduction¶

Let \(P_1\) and \(P_2\) be 2 problems.

Suppose that if we find an algorithm that solves \(P_2\), then we have a solution of \(P_1\). Then \(P_1\) is said to reduce to \(P_2\).

Example

\(P_1\): is \(x\) the square root of \(y\)?

\(P_2\): is \(x\) the 4th root of \(y\)?

If I knew how to solve \(P_2\) in polynomial time, then I can also solve \(P_1\) in polynomial time.

\[x = \sqrt y \iff x = \sqrt[4]{y^2}\]

So, if we know how to solve \(P_2\), we can solve \(P_2\) by squaring the input \(y\) and feeding it into \(P_2\).

Put another way, in functional form:

\[P_1(x,y) = P_2(x,y^2)\]

Therefore, \(P_1\) reduces to \(P_2\); \(P_1 \propto P_2\)

This is technically called a polynomial-time 1-1 reduction, basically the transformation from \(P_1\) to \(P_2\) takes polynomial-time. This is the only type of reduction we will look at.

Definition

Reduction

Let \(P_1\) and \(P_2\) be decision problems.

\(P_1 \propto P_2\) means that:

any instance of \(P_1\) can be transformed in polynomial time into an instance of \(P_2\) in such a way that yes/no answers are preserved.

Properties of Reduction¶

  • \(P_1 \propto P_1\)
  • \(P_1 \propto P_2\) and \(P_2 \propto P_3 \Rightarrow P_1 \propto P_3\)
    • transitive property of reduction
    • this is relly useful - we can solve a bunch of problems at once!

Partition problem:¶

Given a set of integers, \(S\), does \(S\) have a subset that sums to \(\frac{1}{2}\) the total sum?

Subset sum:¶

Given a set of integers, \(S\), and taget value \(k\), does \(S\) have a subset that sums to \(k\)?

Claim:¶

Partition problem \(\propto\) Subset sum problem

Proof:¶

Let \(S\) be an instance of of Partition

Let \(S, \sum_{s \in S} \frac{S}{2}\) be the constructed Subset sum instance.

This transformation is \(O(n)\), and answer preserving.

Therefore, Partition \(\propto\) Subset Sum

Fact

Subset Sum also reduces to Partition! - think about it! Solution on Friday.

Cooke and Levin, independently, searched for the hardest problem in \(NP\) - one to which all others could be reduced.

Can we find such a problem, \(X \in NP\) such that all problems in \(NP\) reduce to \(X\)? - the holy grail of problems.

Definition

Conjunctive Normal Form Satisfiability (CNF-SAT) problem

  • a CNF is a boolean expression structured as a set of clauses joined by “and”, and containing only literals and “or“

    • Example

      \((X_1 + X_2 + \bar X_3) * (\bar X_1 + \bar X_2) * (X_1 + X_3 + X_4)\)

  • SAT means: can it be true? i.e. does it not contain any contradictions?

The problem is to determine whether or not a given CNF is satisfiable.
