Graph problems¶

Problem 1¶

You have a map of communication routes between nodes. Each route has a bandwidth value. Find the path from A to B such that the minimum bandwidth on a route is maximized.

Problem 2¶

You have a maze of rooms connected with passages. Each room has some amount of gold in it. Find a route from A to B that minimizes travel time while maximizing gold collected. Cannot not revisit the same room.

Problem 3¶

You have a travel network of flights. Each flight has a departure time and an arrival time. Find a route from A to B that minimizes travel time.

Dijkstra’s Algorithm¶

This algorithm is the solution to the shortest path problem:

Given a weighted graph, with all weights greater than zero, find the shortest path from A to all other connected nodes.

  1. Let \(A\) be our starting node.
  2. Let \(D(X)\) be the distance variable from \(A\) to connected node \(X\).
  • \(D(A)=0\) and \(D(X)=\infty\) for all others
  1. Let \(w(X, Y)\) be the given distance between joined nodes \(X\) and \(Y\).
  2. Let \(P(X)\) be the variable to track the predecessor in the shortest path to the node \(X\).
  3. Let \(Q\) be the set of nodes we can reach in one step. Initially, \(Q={A}\)
  4. While \(Q\) is non-empty, do the following:
  1. Choose \(X\), the node in math:Q with the minimum distance from where you are.
  2. Then, for all \(Y\), not yet reached neighbours of \(X\), if \(D(X) + W(X, Y) < D(Y)\) then let \(D(Y) = D(X) + w(X,Y)\) and let \(P(Y) = X\).

Note

this last condition will always be true the first time Y is visited. (since \(D(Y)=\infty\))

  1. If \(Y \notin Q\), add \(Y\) to \(Q\), remove \(X\) from \(Q\), mark \(X\) reached.
  2. repeat from step i. so long as condition 6. is met.
