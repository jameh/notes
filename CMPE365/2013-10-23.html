Dynamic Programming for Subset Sum¶

Give \(S=`set of integers :math:`=\{s_1,s_2,...,s_n\}\), target value \(k\).

\(T(s_i,x):=`True iff there exists a subset of :math:\){s_1,s_2,...,s_i}` that sums to \(x\).

if there is such a subset, it either includes \(s_i\), or it doesn’t.

if it does include \(s_i\), then \(\{s_1,...,s_{i-1}\}\) has a subset that sums to \(x-s_i\)

if it doesn’t, then \(\{s_1,...,s_{i-1}\}\) has a subset that sums to \(x\)

Example

\[\begin{split}S=\{1,3,5,8,12\}\\ k=14\end{split}\]

So there is a subset that sums to 14: \(\{8,5,1\}\)

Size of input is \(=n+1\)

Size of table is \(n\cdot k\)

So, alg \(\in O(n\cdot k)\)

Dynamic Programming for Longest Common Subsequence¶

Suppose we have a partial DNA sample e.g.

...GATT...CA...TAA...

Catalogue:

  1. xxGxxATTCxx

Given \(S_1,S_2\), strings of characters, find the longest sequence of characters that is a subsequence of \(S_1\) and \(S_2\).

\(S_1\): “caller”

\(S_2\): “clearer”

. c l e a r e r
c 1 1 1 1 1 1 1
a 1 1 1 2 2 2 2
l 1 2 2 2 2 2 2
l 1 2 2 2 2 2 2
e 1 2 3 3 3 3 3
r 1 2 3 3 4 4 4

Cell :math:`(l_2,r_1)`contains the length of the longest common subsequence of “call” and “clear”

\[T(i,j)=|l.c.s.|\text{ of }S_1[1...i]\text{ and }S_2[1...j]\]

if \(S_1[i]==S_2[j]\), \(T(i,j)=1+T(i-1,j-1)\)

else \(T(i,j)=max\{T(i,j-1),T(i-1,j),T(i-1,j-1)\}\)

“cler” or “caer”
