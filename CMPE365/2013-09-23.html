k-clique \(\in\) NP-Complete¶

Transforming CNF-SAT problem into k-clique problem:

\(E = (x_1 + x_2 + \bar x_3) * (\bar x_1 + \bar x_2) * (x_1 + \bar x_2 + x_3)\)

In each term of the product of sums, one of the summands must have a value of True.

The lines of the graph were drawn such that the nodes they connect are mutually satisfiable - i.e. do not contain a contradiction.

We need a bijective transformation from CNF-SAT to k-clique.

Suppose \(E\) is satisfiable. Then each clause has a True literal flag, and the vertices corresponding to those clauses must form a k-clique in the graph.

Now suppose the graph \(G\) has a k-clique. Then the k-clique must contain 1 vertex from each vertical set. (since there are no connections within a vertical set by construction). And, again by construction, since they are mutually adjacent in the graph, they must not contradict each other.

Therefore, since a polynomial-time transformation exists from CNF-SAT to k-clique, k-clique is NP-Complete.

Partition & Subset-Sum¶

Partition¶

Given a set of integers, is there a subset that sums to exactly \(\frac{1}{2}\) the total?

Subset-Sum¶

Given a set of integers, and a target integer, \(k\), is there a subset that sums to \(k\)?

It is clear that every instance of Partition is also an instance of Subset-Sum (let \(k=\frac{1}{2}\)). Therefore, Partition \(\propto\) Subset-Sum.

In general, special cases always reduce to general cases.

Does Subset-Sum \(\propto\) Partition?¶

Given an instance of Subset-Sum, \(S= \{...\}\), \(k\). (Can we split \(S\) into \(S_1 = \{...\}\), \(S_2 = \{...\}\) s.t. \(S_1\) sums to \(k\)?)

Try to construct \(S'\) so that \(S'\) has a partition iff \(S\) has a subset summing to \(k\).

Try adding some value \(x\) to \(S\).

\(S' = \{s_1, ... s_n, x\}\)

split \(S'\) into \(S_1'\) and \(S_2'\) such that they have the same sum.

\(S_1' = \{..., x\}\) \(S_2' = \{\text{all from } S\}\)

Let \(T = \sum_S\)

if \(S\) has a subset summing to \(k\),

\(S_1 = \{...\}\) sums to \(k\), \(S_2 = \{...\}\) sums to \(T-k\).

So let \(T-k +x = k\) to make them equal partitions. Equivalently, \(x = 2k - T\).

So with out Subset-Sum problem \((S, k)\), construct \(S' = S \cup \{2k-T\}\) where \(T=\) total sum of S.

Then \(\sum S' = \sum S + 2k - T = 2k\)

This transformation is bijective, and this shows that Subset-Sum \(\propto\) Partition.
