Cook-Levin¶

Theorem

If CNF-SAT \(\in\) P, then \(P=NP\)

Proof would be to show that any decision problem in \(NP\) can be transformed in polynomial time to an instance of CNF-SAT. i.e., all prolems in \(NP\) reduce to CNF-SAT.

NP-Complete Problems¶

Problem \(X\) is NP-Complete if \(X \in NP\) and (\(Y \propto X\) or \(Y \in P\))

There are many NP-Complete problems.

Given a new problem, \(Z\), we can show that \(Z\) is NP-Complete.

If we know problem \(X\) is NP-Complete, all we need to show is that \(X \propto Z\).

Example

Show k-clique is NP-complete.

Given graph \(G\), \(k \in \Bbb Z_{\geq 0}\), does \(G\) contain a complete subgraph of size \(k\)?

We will show that CNF-SAT \(\propto\) k-clique.

Suppose \(E = (x_{1,1} * x_{1,1} * ...) + (x_{2,1} * x_{2,1} * ...) + ... + (x_{n,1} * x_{n,1} * ...)\) is the given instance of CNF-SAT.

Construct an instance of k-clique as follows:

  1. Construct a vertex set for each clause of E - one vertex for each literal, labelled with the literal.
  1. Join all vertices that are in different sets and are not contradictory.
  2. Choose k. Let k be the number of clauses in E.
