Liouville Theorem, Cauchy Differentiation Formula¶

Proof of Theorem 1 (For first derivative only)

\[G(z)=\frac{1}{2πi}\int_γ\frac{g(w)}{w-z}dw\]

(\(z\in \mathbb C\backslash γ\))

\[\begin{split}&\frac{G(z+h)-G(z)}{h}\\ =&\frac{1}{h}\frac{1}{2πi}\int_γ\frac{g(w)}{w-(z+h)}-\frac{g(w)}{w-z}dw\\ =&\frac{1}{h}\frac{1}{2πi}\int_γ\frac{hg(w)}{(w-(z+h))(w-z)}dw\\ =&\frac{1}{2πi}\int_γ\frac{g(w)}{(w-(z+h))(w-z)}dw\end{split}\]

We want to show this converges to

\[\frac{1}{2πi}\int_γ\frac{g(w)}{(w-z)^2}dw\]

as \(h\to 0\).

Well, \(|(w-(z+h))(w-z)^2|\geq δ^3\), so

\[\begin{split}&\frac{1}{2πi}(\int_γ\frac{g(w)}{(w-(z+h))(w-z)}dz-\int_γ\frac{g(w)}{(w-z)^2}dz)\\ =&\frac{1}{2πi}\int_γ\frac{hg(w)}{(w-(z+h))(w-z)^2}dw\end{split}\]

By uniform convergence argument.

Let \(L=length(γ)\) and \(M=max_{w\in γ}|g(w)|\), then:

\[\begin{split}&|\frac{1}{2πi}\int_γ\frac{hg(w)}{(w-(z+h))(w-z)^2}dw|\\ \leq &\frac{h}{2π}\frac{LM}{δ^3}\to_{h\to 0}\to 0\end{split}\]

Giving results.

Argument for higher derivatives follows similarly from induction.

Corollary 1 (A holomorphic function is infinitely differentiable.)

Proof:

If \(f:D\to\mathbb C\) is holomorphic on some open set, then \(\forall z_0\in D\), \(\exists ρ>0\) s.t. \(D_ρ(z_0)\subset D\) and disks are simple connected. Use CDF.

Corollary 2 (Cauchy’s Inequality for Derivative)

If \(f:D\to\mathbb C\) is holomorphic and a circle \(γ\subset D\) has radius \(R\) centered at \(z_0\) and \(max_{w\in γ}|f(z)\leq M|\) then,

\[|f^{(k)}(z_0)|\leq \frac{k!}{R^k}M\]

Proof:

\[\begin{split}|f^{(k)}(z_0)|=|\frac{k!}{2πi}\int_γ\frac{f(w)}{(w-z)^{k+1}}dw|\\ \leq \frac{k!}{2πi}\frac{M}{R^{k+1}}2πR\\ =\frac{k!M}{R^k}\end{split}\]

Corollary 3 (Liouville’s Theorem)

The only bounded entire functions are constant.

Proof:

If \(f:\mathbb C\to\mathbb C\) is holomorphic and bounded, \(\exists M>0\) s.t. \(|f(z)|\leq M\) for all \(R>0\), \(\forall z\in \mathbb C\) since we can take \(γ\) arbitrarily large.

So \(f'(z)=0\) for all \(z\in\mathbb C\). So \(f(z)\) is constant.

Corollary 4: (Fundamental Theorem of Algebra) (FTA)

Ever Polynomial on \(\mathbb C\) factors completely into linear components

Proof:

If \(P(z)=a_nz^n+...+a_1z+a_0\) is a polynomial of degree \(n\geq 1\), then if \(P(α)=C\), \(P(z)-C=g(z)(z-α)\) where \(g(z)\) is a polynomial of degree \(n-1\).

Why? Polynomial long-division

So it is enough to show \(P(α)=0\) for some \(α\in\mathbb C\), then \(P(z)=q(z)(z-α)\) and we have reduced the problem to degree \(n-1\) and we can proceed by induction until \(g(z)\) is degree 1 (linear).

So FTA \(\iff\) \(P(α)=0\) for \(α\in\mathbb C\).

Proof by Contradiction:

Suppose \(p(α)\neq 0\) for all \(α\in \mathbb C\). Then \(f(z)=\frac{1}{P(z)}\) is entire.

Since \(|P(z)|\geq |a_n||z|^n-|a_{n-1}||z|^{n-1}-...-|a_1||z|+|a_0|\),

We have

\[|P(z)|\geq \frac{|a_n|}{2}|z|^n\]

for sufficiently large \(|z|\geq ρ>0\)

So

\[|f(z)|=\frac{1}{|P(z)|}\leq \frac{2}{|a_n||z|^n}\leq \frac{2}{p^n|a_n|}\]

for \(|z|\geq ρ\).

and \(M=max_{|z|\leq ρ}|f(z)|\) exists since \(f(z)\) is continuous everywhere and \(|z|\leq ρ\) is compact so:

\[|f(z)|\leq max(M,\frac{2}{ρ^n|a_n|})\]

so \(f(z)\) is bounded, thus constant.
