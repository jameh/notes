Max Modulus Theorem & Harmonic Functions¶

Proof of Local Max Modulus¶

Idea:

If \(f:D_ρ(z_0)\to\mathbb C\) is holomorphic and \(|f(z)|\) has a local max at \(z_0\), then \(|f(z)|\) is constant on \(D_ρ(z_0)\).

So suppose \(|f(z_0)|>|f(z)|\) for some \(z\in D_ρ(z_0)\). Let \(M=|f(z_0)|\) and \(r=|z-z_0|\). Let \(z=z_0+re^{ia}\).

By continuity, \(\exists δ\) s.t. \(\forall ε>0\) small enough,

\[\begin{split}|f(z_0+re^{iθ})|<|f(z_0)|-ε\end{split}\]

for \(θ\in (a-δ,a+δ)\).

But then

\[\begin{split}M&=|f(z_0)|\\ &=\left|\frac{1}{2πi}\int_0^{2π}f(z_0+re^{iθ}dθ)\right|\\ &=\frac{1}{2π}\left|\int_{a-δ}^{a+δ}f(z_0+re^{iθ})dθ+\int_{\text{rest}}f(z_0+re^{iθ})dθ\right|\\ &\leq \frac{1}{2π}((M-ε)(2δ)+M(2π-2δ))\\ &=M-\frac{δε}{π}\end{split}\]

and we have reached our contradiction.

So strict inequalities don’t hold with local max.

So \(|f(z_0)|=|f(z)|\) for all \(z\in D_ρ(z_0)\) and so \(|f(z)|\) is constant. Thus, so is \(f(z)\).

Theorem (Max Modulus Theorem, Global)

Suppose \(f:\bar D\to\mathbb C\) is holomorphic on the bounded domain \(D\subset S\) and continuous on \(\bar D\), the closure of \(D\).

Let \(M=max_{z\inδD}|f(z)|=\) max. modulus on the boundary of \(D\).

Then:

  1. \(|f(z)|\leq M\) \(\forall z\in \bar D\)
  2. If \(|f(z)|=M\) for some \(z\in D\), then \(f(z)\) is constant on \(\bar D\).

Proof:

Let \(M'=max_{z\in \bar D}|f(z)|\). Suppose \(|f(z_0)|=M'\) for some \(z_0\in D\).

Then \(z_0\) is a max and for every \(ρ>0\) s.t. \(D_ρ\subset D\), \(|f(z)|\) is constant on \(D_ρ(z_0)\) by local Max Modulus Theorem.

Since \(D\) is path connected, \(\forall z\in D\), \(\exists γ:[0,1]\to D\) s.t.

\[γ(0)=z_0\]

and

\[γ(1)=z\]

where \(γ\subset D\) is compact, and so it has a minimal distance, \(ε>0\), from the boundary of \(D\).

Since \(γ\) has finite length, we can cover it with disks of radius \(\frac{ε}{2}\) so that the center of each disk is in the previous disk.

and we can do this with finitely many disks. By local Max Modulus Thm, \(f(z_0)\) is constant on the first disk and \(|f(z)|=M'\) on first disk. Since \(z_1\) is in the first disk, \(|f(z_1)|=M'\) and local Max Modulus again gives \(f(z)\) is constant on the second disk. Repeating this argument finitely many times gives that \(f(z)\) is constant on \(γ\).

So \(f(z_0)=f(z)\) and so \(f(z)\) is constant on \(D\), and \(|f(z)|=M'\) on \(D\).

Now \(f\) is continuous on \(\bar D\) - what does continuity mean on a closed set?

\(\forall ε>0\), \(\exists d>0\) s.t. if \(|z-w|<δ\) and \(z,w\in\bar D\), then \(|f(z)-f(w)|<ε\). So this gives continuity up to boundary.

Recall

Boundary points are the \(z_0\in\bar D\) such that \(D_ρ(z_0)\cap D\neq \{\}\) and \(D_ρ(z_0)\cap (\mathbb C\backslash D)\neq\{\}\) \(\forall ρ>0\).

If \(f(z)\) is constant on \(D\) and continuous on \(\bar D\), then \(f(z)\) is constant on \(\bar D\), since

\[\begin{split}|f(z)-f(z_0)|<ε\end{split}\]

\(\forall z\in δD\)

So \(|f(z)|=M'\) on \(δD\), but \(max_{z\in δD}|f(z)|=M\). So \(M=M'\). This proves (a), \(|f(z)|\leq M\), \(\forall z\in \bar D\) and (b) since if that max is achieved \(f(z)\) must be constant on \(\bar D\).

Now we want to prove all this for Harmonic Functions.

Theorem (Mean Value Property for Harmonic Functions)

\(D\) containing a closed disk \(D_ρ(x_0,y_0)\), then

\[u(x_0,y_0)=\frac{1}{2π}\int_0^{2π}u(x_0+r\cos t,y_0+r\sin t)dt\]

Proof:

Since \(D_ρ(x_0,y_0)\subset D\), \(\exists\) a simply connected open set \(\tilde D\) s.t. \(D\supset\tilde D\supset D_ρ(x_0,y_0)\) and by the reconstruction theorem, \(\exists v:\tilde D\to \mathbb R\) s.t. \(f=u+iv\) is analytic on \(\tilde D\). Thus \(f\) has the mean value property:

\[f(z_0)=\frac{1}{2π}\int_0^{2π}f(z_0+re^{it})dt\]

where \(z_0=x_0+iy_0\)

So

\[\begin{split}u(x_0,y_0)+iv(x_0,y_0)&=\frac{1}{2π}\int_0^{2π}u(x_0+r\cos t,y_0+r\sin t)+iv(x_0+r\cos t,y_0+r\sin t)dt\\ &=\frac{1}{2π}\int_0^{2π}u(x_0+r\cos t,y_0+r\sin t)dt +i\frac{1}{2π}\int_0^{2π}v(x_0+r\cos t,y_0+r\sin t)dt\end{split}\]

Comparing real and imaginary parts, we are done.

Theorem (Local Max/Min principle for Harmonic Functions)

Note

There’s no modulus for harmonic functions, they are real-valued.

If \(U:D\to\mathbb R\) is harmonic and \(\exists\) a local max/min at \((x_0,y_0)\in D\), i.e.

\[u(x_0,y_0)\geq u(x,y)\]

or

\[u(x_0,y_0)\leq u(x,y)\]

\(\forall (x,y)\in D_δ(x_0,y_0)\)

then \(U\) is constant on an on an open disk around \((x_0,y_0)\).

Proof:

For local Max, same as for holomorphic functions, just need Max Value Principle and Continuity. For local min, use local Max argument on \(-u(x,y)\).

Theorem (Global Max/Min principle for Harmonic Functions)

\(U: D\to\mathbb R\) is harmonic on \(D\), continuous on \(\bar D\).

Let

\[M=max_{(x,y)\in δD}u(x,y)\]

and

\[m=min_{(x,y)\in δD}u(x,y)\]

then

  1. \(m\leq u(x_0,y_0)\leq M\) \(\forall (x_0,y_0)\in\bar D\).
  2. If \(u(x,y)=M\) or \(m\) for some \((x,y)\in D\), then \(u\) is constant on \(D\).

Proof:

Same as for max modulus, just use local Max/Min principle instead of Local Max Mod.
