We will integrate along paths.

For \(g:(a,b)\subseteq\mathbb R^2\to\mathbb C\)

where \(g(t)=u(t)+iv(t)\), we define:

\[\int_a^b g(t)dt=\int_a^bu(t)dt+i\int_a^bv(t)dt\]

Example

\[f:(0,\frac{\pi}{4})\to\mathbb C\]\[f(t)=cos(t)+isin(t)\]\[\int_0^{\frac{\pi}{4}}f(t)dt=\int_0^{\frac{\pi}{4}}\cos tdt+i\int_0^{\frac{\pi}{4}}\sin t dt\]\[=[\sin t]|^{\frac{pi}/{4}}_0+i[-\cos t]|^{\frac{pi}/{4}}_0\]\[=\frac{\sqrt(2)}{2}+i(-\frac{\sqrt(2)}{2}+1)\]

but \(f(t)=e^{it}\)

So, one may wonder,

\[\int_0^{\frac{\pi}{4}}e^{it}dt=^?[\frac{1}{t}e^{it}]{\frac{\pi}{4}}\]

well,

\[=\frac{1}{i}(\frac{\sqrt{2}}{2}+i\frac{\sqrt{2}}{2}-1)\]\[=\frac{\sqrt{2}}{2}+i(1-\frac{\sqrt{2}}{2})\]

Recall

That an oriented (directed) curve \(\gamma:[a,b]\to\mathbb C\) is said to be smooth pr \(C^1\) if

\[\gamma(t)=x(t)+iy(t)\]

and

\[\gamma'(t)=x'(t)+iy'(t)\]

exists, and is continuous.

A curve is piecewise-smooth if it finitely divides into smooth pieces

If \(f:S\to\mathbb C\) is a complex-valued function (not necessarily analytic) and \(\gamma\subset S\) is a curve, we want to define

\[\int_\gamma f(z)dz\]

How do we do this? -Riemann Sums.

consider the sum

\[\sum_{k=1}^n f(z_k)(z_k=z_{k-1})\]

If \(\gamma:[a,b]\to\mathbb C\), we can partition \([a,b]\) so that

\[\gamma(t_k)=z_k\]

Rewrite the sum as:

\[\sum_{k=1}^n f(\gamma(t_k))(\gamma(t_k)=\gamma(t_{k-1}))\]\[=\sum_{k=1}^n f(\gamma(t_k))\frac{(\gamma(t_k)=\gamma(t_{k-1}))}{t_k-t_{k-1}}(t_k-t_{k-1})\]

Taking the limit of the partition as \(n\to\infty\), the sum becomes:

\[\int_a^b f(\gamma(t))\gamma'(t)dt\]\[=: \int_\gamma f(z)dz\]

Example

f(z)=frac{1}{z}

gamma(t)=cos(t)+isin(t)

for \(t\in [0,2\pi]\)

\[\int_\gamma f(z)dz=\int_0^{2\pi}f(\gamma(t))\gamma'(t)dt\]\[=\int_0^{2\pi}\frac{1}{\cos t+i\sin t}(-\sin t+i\cos t)dt\]

(and since \(\frac{1}{\cos t+i\sin t}=e^{-it}=\cos t-i\sin t\))

\[=\int_0^{2\pi}(\cos t-i\sin t)(-\sin t+i\cos t)\]\[=i\int_0^{2\pi}1dt\]\[=2\pi i\]
