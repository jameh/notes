Limits, Continuity, Differentiability¶

Recall

Prop.1: Limits are unique if they exist

Proposition 2

Suppose \(\lim_{z \to z_0} f(z) = a\) and \(\lim_{z \to z_0} g(z) = b\)

  1. \(\lim_{z \to z_0} (f(z) + g(z)) = a+b\)
  2. \(\lim_{z \to z_0} f(z)g(z) = ab\)
  3. \(\lim_{z \to z_0} f(z)/g(z) = a/b\) if \(b \neq 0\)

Proof:

  1. We know
\[|f(z)g(z) -ab| = |f(z)g(z) - ag(z) + ag(z) - ab|\]\[\leq |g(z)||f(z) - a| + |a||g(z)-b|\]

by the triangle inequality.

then,

\(\forall \epsilon > 0\), \(\exists \delta_1, \delta_2\) such that for

\[\begin{split}|z-z_0| < min(\delta_1, \delta_2) = \delta\end{split}\]\[\begin{split}|g(z) -b| < min(\epsilon/(2(1+|a|)), 1)\end{split}\]\[\begin{split}|f(z) - a| < \epsilon/(2(1 + |a|))\end{split}\]\[\leq (1+|a|)(\epsilon/(2(1+|a|))) + |a|\epsilon/(2(1+|a|))\]\[\begin{split}< \epsilon/2 + \epsilon/2 = \epsilon\end{split}\]

So, \(\lim_{z \to z_0}f(z)g(z) = ab\)

Continuity¶

A function \(f: S \subseteq \Bbb C \to \Bbb C\), where \(S\) is open, is continuous at \(z\) if \(\lim_{z \to z_0}f(z) = f(z_0)\).

\(S\) must be open because every point needs an open disks around it to take the complex limit.

f is continuous on \(S\) if it is continuous for all \(z_0 \in S\).

the \(\epsilon-\delta\) definition:

\[\begin{split}\forall \epsilon > 0, \exists \delta \text{ s.t.}\end{split}\]\[\begin{split}|f(z) - f(z_0)| < \epsilon \text{ if } |z - z_0| < \delta\end{split}\]

Not punctured disks anymore.

Proposition 3

If \(f(z)\) and \(g(z)\) are coninuous on the open set \(S \subseteq \Bbb C\), Then so are:

  1. \(f+g\)
  2. \(fg\)
  3. \(f/g\) when \(g(z) \neq 0\)

Proof: use continuouty definition and Prop. 2

Corollary 1

All polynomials and rational functions are continuous.

polynomial example:

\(f(z) = a_nz^n + ... + a_1z + a_0\)

rational function example:

\(f(z) = \frac{a_nz^n + ... + a_1z + a_0}{b_nz^n + ... + b_1z + b_0}\)

Proof:

every rational function and polynomial can be built using \(f(z) = z\) and \(f(z) = constant\). Then use Prop. 3.

Proposition 4

  1. If \(\lim_{z \to z_0} f(z) = a\)

and \(g(z)\) is continuous at \(a\), then

\[\lim_{z \to z_0}g(f(z)) = g(a)\]
  1. If \(A,B\) are open sets in \(\Bbb C\) (\(f: A \to \Bbb C\) and \(g: B \to \Bbb C\)) are continuous, and \(f(A) \subseteq B\) then \(g(f(z))\) is continuous on \(A\).

Complex Differentiability¶

Let \(S \subseteq \Bbb C\) be open.

A function \(f: S \to \Bbb C\) is (complex) differentiable at \(z_0 \in S\) if the limit:

\[\lim_{z \to z_0} (f(z) - f(z_0))/(z-z_0) = \lim{h \to 0} (f(z_0 + h) - f(z_0))/h\]

exists.

The limit is denoted \(f'(z_0)\) or \(\frac{df}{dz}z_0\)

\(f\) is said to be holomorphic or analytic on \(S\) if it is differentiable for every \(z_0 \in S\).

\(f\) is entire if it is holomorphic on the all of \(\Bbb C\).

Is this definition strange?

when we had a function

\[F: \Bbb R^2 \to \Bbb R^2\]

that is differentiable, what does that mean?

Means the Jacobian exists.

Proposition 5

If \(f\) is differentiable at \(z_0\), it is continuous at \(z_0\).

Proof:

We have that the limit

\[\lim_{z \to z_0}\frac{f(z)- f(z_0)}{z - z_0}\]

exists, so we must have that the numerator limit exists.

Example

Is \(f(z) = z\) entire?

Pick \(z_0 \in \Bbb C\)

\[\lim_{z \to z_0} \frac{f(z) - f(z_0)}{z - z_0}\]\[= \lim_{z \to z_0} \frac{z - z_0}{z - z_0} = 1\]\[f'(z) = 1\]

So, yes!

Example

is \(f(z) = C\) entire? \(C \in \Bbb C\)

\[f(z) = 0\]

everywhere

Example

is \(f(z) = \bar z\) entire?

take the limit:

\[\lim_{h \to 0} \frac{f(z_0+h)- f(z_0)}{h}\]\[= \lim_{h \to 0}\frac{\bar z_0 + \bar h - \bar z_0}{h}\]\[= \lim_{h \to 0} \frac{\bar h}{h}\]\[= \lim_{|h| \to 0} e^{-2i \theta}\]

limit does not exist anywhere.
