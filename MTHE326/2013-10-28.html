Winding Numbers¶

Definition

A winding number (or index) counts the number of times a closed loop goes around a point.

If \(γ\) is a closed curve around \(z_0\in\mathbb C\), \(z_0\notin γ\), then we define the index by

\[I(γ,z_0)=\frac{1}{2πi}\int_γ\frac{1}{z-z_0}dz\]

Proposition

The index is always an integer.

Let \(γ=γ(s)\) for \(s\in [0,1]\) and let

\[g(t)=\int_0^t\frac{γ'(s)}{γ(s)-z_0}ds\]

By FTC (for real functions),

\[g'(t)=\frac{γ'(s)}{γ(s)-z_0}\]

so,

\[\frac{d}{dt}e^{-g(t)}(γ(t)-z_0)=0\]

(which we verify by taking real and imaginary parts) so,

\[e^{-g(0)}(γ(0)-z_0)=e^{-g(1)}(γ(1)-z_0)\]

but \(γ(0)=γ(1)\neq z_0\)

So \(e^{-g(0)}=e^{-g(1)}\). But \(g(0)=0\) so \(e^{-g(1)}=1\)

Thus, \(g(1)=2πik\) for \(k\in\mathbb Z\).

So,

\[\frac{1}{2πi}g(1)=\frac{1}{2πi}\int_γ\frac{1}{z-z_0}dz=k\in\mathbb Z\]

Neat, not very surprising. More surprising:

Theorem

Cauchy’s Integral Formula (Part 1)

Suppose that \(D\) is simply connected and \(f:D\to\mathbb C\) is holomorphic. If \(γ\) is a closed curve winding once around \(z_0\in D\), \(z_0\notin γ\), then

\[f(z_0)=\frac{1}{2πi}\int_γ\frac{f(z)}{z-z_0}dz\]

why is this special?

  1. \(f(z)\) is completely determined by its behaviour on \(γ\) (not that surprising if you know Green’s Theorem)
  2. The value of \(f\) is obtained by integrating \(f\).

Proof:

It is not hard to see that \(γ\) is homotopic to \(γ_r\) in \(\mathbb C\backslash \{z_0\}\), a counter-clockwise circle of radius \(r\) centered at \(z_0\) (for \(r\) small enough)

So,

\[\frac{1}{2πi}\int_γ\frac{f(w)}{w-z_0}dw=\frac{1}{2πi}\int_{γ_r}\frac{f(w)}{w-z_0}dw\]

by Cauchy’s Theorem.

We already know:

\[\frac{1}{2πi}\int_{γ_r}\frac{1}{w-z_0}dw=\frac{1}{2πi}\cdot 2πi=1\]

So,

\[\frac{1}{2πi}\int_{γ_r}\frac{f(z_0)}{w-z_0}dw=f(z_0)\]

So we need to show

\[\frac{1}{2πi}\int_{γ_r}\frac{f(w)}{w-z_0}dw=\frac{1}{2πi}\int_{γ_r}\frac{f(z_0)}{w-z_0}dw\]

Or, equivalently:

\[\frac{1}{2πi}\int_{γ_r}\frac{f(w)-f(z_0)}{w-z_0}dw=0\]

We know

\[|\frac{1}{2πi}\int_{γ_r}\frac{f(w)-f(z_0)}{w-z_0}dw|\leq |\frac{1}{2πi}|\cdot max_{w\in γ_r}|\frac{f(w)-f(z_0)}{w-z_o}|\cdot 2πr\]

Also, \(|w-z_0|=r\) for \(w\in γ_r\).

Thus,

\[|\frac{1}{2πi}\int_{γ_r}\frac{f(w)-f(z_0)}{w-z_0}dw|=M_r:=max_{w\in γ_r}|f(w)-f(z_0)|\]

But \(f\) is continuous, so for all \(ε>0\), \(\exists δ>0\) s.t. \(|f(w)-f(z_0)|<ε\) if \(|w-z_0|<δ\). If we make \(r\) small enough that \(γ_r\subset D_δ(z_0)\) then \(M_r<ε\), so

\[\begin{split}|\frac{1}{2πi}\int_γ\frac{f(w)-f(z_0)}{w-z_0}dw|<ε\end{split}\]

for all \(ε>0\), thus 0. Giving proof.

Example

\(γ\) winds around 3 once counterclockwise, then:

\[\frac{1}{2πi}\int_γ\frac{z^3+z-2}{z-3}dz=28\]

Integrals of “Cauchy type”¶

If \(g:γ\to\mathbb C\) is continuous and \(γ\subset \mathbb C\) is a curve, then we define

\[G(z_0)=\frac{1}{2πi}\int_γ\frac{g(w)}{w-z_0}dw\]

for \(z_0\notin γ\)

Claim: \(G: \mathbb C\backslash γ\to \mathbb C\) is holomorphic.

Theorem 1 (Cauchy-Type Integrals are analytic)

If \(γ\) is a curve (not necessarily a loop) and \(g:γ\to\mathbb C\) is continuous then:

\[G(z)=\frac{1}{2πi}\int_γ\frac{g(w)}{w-z}dw\]

is analytic on \(\mathbb C\backslash γ\) and the kth derivative is given by:

\[g^{(k)}(z)=\frac{k!}{2πi}\int_γ\frac{g(w)}{(w-z)^{k+1}}dw\]

Heuristic idea: See Homework Problem 1

Proof: Later.

Theorem (Cauchy’s Differentiation Formula) (CDF)

If \(f:D\to\mathbb C\) is analytic on simply connected \(D\), then \(f\) is infinitely differentiable on \(D\) and for each \(z_0\in D\),

\[f^{(k)}(z)=\frac{k!}{2πi}\int_γ\frac{f(w)}{(w-z^{k+1})}dw\]

for any closed loop \(g\subset D\) with \(I(γ,z_0)=1\)

Proof: follows from part 1 and analyticity of Cauchy Type Integrals.

Example

\[\begin{split}\int_{|z|=1}\frac{e^z}{z^3}dz&=\frac{2πi}{2!}(\frac{2!}{2πi}\int_{|z|=1}\frac{e^z}{(z-0)^3}dz)\\ &=πi(\frac{d^2}{dz^2}|_0)\\ &=πi(e^0)=πi\end{split}\]
