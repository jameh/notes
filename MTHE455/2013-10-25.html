An Example of solving a Stationary Distribution¶

Example

We have \(n\) processors. Processor \(i\) will successfully complete a job with probability \(p_i\), and fail with probability \(1-p_i\), \(i=1,...,n\).

The processors are placed in some order. When a job arrives to the system, it tries the processors in order until it is successfully completed, or until all processors fail.

If the \(k^{th}\) processor is the first to be successful, it exchanges its place with the processor ahead of it. The ordering does not change if the first processor is successful or if all processors are unsuccessful.

Let \(X_n=\) ordering of the processors after the \(n^{th}\) job. \(\{X_n, n\geq 0\}\) is a Markov chain with state space being the set of all orderings (permutations) of \(\{1,...,n\}\). Let us compute the transition probabilities.

Let \(i=(i_1,i_2,...,i_n)\) be an arbitrary ordering in the state space.

Let \(j_k\) denote state \(i\) but with components \(k\) and \(k+1\) interchanged, \(k=1,...,n-1\)

Then

\[p_{i,j_k}=(1-p_{i_1})(1-p_{i_2})...(1-p_{i_k})p_{k+1}\]

for \(k=1,...,n-1\)

\[p_{i,i}=p_{i_1}+(1-p_{i_1})...(1-p_{i_n})\]

We need \(p_{j_k,i}\):

\[\begin{split}p_{j_k,i}=\begin{cases} (1-p_{i_k})...(1-p_{i_{k-1}})(1-p_{i_{k+1}})p_{i_k} & k=2,...,n-1\\ (1-p_{i_2}p_{i_1}) & k=1 \end{cases}\end{split}\]

The local balance equations are:

\[Π_ip_{i,j_1}=Π_{j_1}p_{j_1,i}\]

is

\[Π_i(1-p_{i_1})p_{i_2}=Π_{j_1}(1-p_{i_2})p_{i_1}\]

and

\[Π_ip_{i,j_k}=Π_{j_k}p_{j_k,i}\]

is

\[Π_i(1-p_{i_1})...(1-p_{i_k})p_{i_{k+1}}=Π_{j_k}(1-p_{i_1})...(1-p_{i_{k-1}})(1-p_{i_{k+1}})p_{i_k}\]

or

\[Π_i(1-p_{i_k})p_{i_{k+1}}=Π_{j_k}(1-p_{i_{k+1}})p_{i_k}\]

which holds for \(k=1\) as well.

These are all the local balance equations that \(Π\) needs to sastisfy.

If we assume that \(Π_i\) is of product form, meaning:

\[Π_i=f_1(p_{i_1})f_2(p_{i_2})...f_n(p_{i_n})\]

where \(f_j\) is a function for each position in the vector \(i=(i_1,...,i_n)\)

For the state \(j_k\) then the value of \(Π_{j_k}\) would be:

\[f_1(p_{i_1})...f_{k-1}(p_{i_{k-1}})f_{k}(p_{i_{k+1}})f_{k+1}(p_{i_k})f_{k+2}(p_{i_{k+2}})...f_n(p_{i_n})\]

Assuming this, we get:

(1)\[f_k(p_{i_k})f_{k+1}(p_{i_{k+1}})(1-p_{i_k})p_{i_{k+1}}=f_k(p_{i_{k+1}})f_{k+1}(p_{i_k})(1-p_{i_{k+1}})p_{i_k}\]

Or try:

(2)\[f_k(p_{i_k})f_{k+1}(p_{i_{k+1}})(1-p_{i_k})=f_k(p_{i_{k+1}})f_{k+1}(p_{i_k})(1-p_{i_{k+1}})\]

After some trial and error, letting \(f_k(p_{i_k})=(1-p_{i_k})^k\), equation (2) is solved. Now, putting back \(p_{i_k}\) and \(p_{i_{k+1}}\) and remove \((1-p_{i_k})\) and \((1-p_{i_{k+1}})\) to get:

(3)\[f_k(p_{i_k})f_{k+1}(p_{i_{k+1}})p_{i_{k+1}}=f_k(p_{i_{k+1}})f_{k+1}(p_{i_k})p_{i_k}\]

And see that \(f_k=(\frac{1}{p_{i_k}})^k\) works for (3), and finally that we can put these two solutions together, giving:

\[f_k(p_{i_k})=(\frac{1-p_{i_k}}{p_{i_k}})^k\]

Which satisfies (1)

Thus,

\[Π_i=C\prod_{j=1}^n(\frac{1-p_{i_j}}{p_j})^j\]

where \(C=(\sum_{i\in S}\prod_{j=1}^n(\frac{1-p_{i_j}}{p_j})^j)^{-1}\)
