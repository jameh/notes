Markov Chain Monte Carlo¶

Let \(S\) be a discrete state space

Let \(Π=(Π_i)_{i\in S}\) be a vector indexed by states (i.e. \(Π\) is a function on \(S\)) such that \(Π_i>0\) for all \(i\in S\) and \(\sum_{i\in S}Π_i<\infty\)

Then we can assume (by dividing by \(\sum_{i\in S}Π_i\)) that \(Π\) is a pmg over \(S\).

Goal:¶

Draw samples from this distribution

Method is to construct a Markov chain that has, as its stationary distribution, \(Π\).

Metropolis-Hastings Algorithm¶

  1. Let \(Q=(q_{ij})_{i,j\in S}\) be the transition matrix of an irreducible Markov chain over \(S\).

    In practice, one should choose \(Q\) so that the Markov chain specified by \(Q\) is not difficult to simulate.

  2. Define the acceptance probabilities \(a_{ij}\)

    (1)\[a_{ij}=min\{1, \frac{Π_jq_{ji}}{Π_iq_{ij}}\}\]

  3. Now let \(p_{ij}=q_{ij}a_{ij}\) for all \(i,j\in S\) (\(i\neq j\)) and \(p_{ii}=1-\sum_{j\neq i}p_{ij}\)

    Then \(P=(p_{ij})_{i,j\in S}\) is the transition matrix of a Markov chain with stationary distribution \(Π\). To see this, we will check that \(Π\) satisfies the local balance equations for this Markov chain.

    Let us adopt the convention that

    \[0\cdot\text{ anything}=0\]

    Fix \(i\neq j\).

    The local balance equation is:

    \[Π_ip_{ij}=Π_jp_{ji}\]

    or

    (2)\[Π_iq_{ij}a_{ij}=Π_jq_{ji}a_{ji}\]

    If \(q_{ij}=q_{ji}=0\), then both sides are 0.

    If \(q_{ij}=0\) and \(q_{ji}>0\), then the LHS of (2) is 0, and \(a_ji=0\) by (1)

    If \(q_{ij}>0\) and \(q_{ji}=0\), then it’s the same as previous case.

    If \(q_{ij}>0\) and \(q_{ji}>0\) then consider 2 cases:

    1. \(Π_jq_{ji}>Π_iq_{ij}\). Then \(a_{ij}=1\) and \(a_{ji}=\frac{Π_iq_{ij}}{Π_jq_{ji}}\)

      Then LHS \(=Π_iq_{ij}\) and RHS is also \(Π_iq_{ij}\)

    2. \(Π_jq_{ji}<Π_iq_{ij}\), same as above.

To implement this, we simulate the Markov chain with transition matrix \(P\). When in state \(i\), choose the next state according to \(Q\).

If state \(j\) is selected, then move to state \(j\) with probability \(a_{ij}\); otherwise state in state \(i\).

Application

Simulated Annealing

Let \(b=(b_i)_{i\in S}\) be a function over \(S\).

Note

\(b_i\) is allowed to be negative

Find the state \(i\in S\) that minimizes \(b_i\).

Form \(Π_i=e^{\frac{-b_i}{T}}\), where \(T>0\) is a parameter called the “temperature”. If \(T\) is large, then \(Π\) will be relatively flat over \(S\). As \(T\) decreases, the value of \(Π_i\), where \(b\) is minimized, gets large relative to other values of \(Π_j\).

Idea

Run Markov chain with stationary distribution (normalized) \(Π\).

Run for a while - keep track of maximum so far

Lower the temerature - run for a while.

Keep repeating until good solution is found.
