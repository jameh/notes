Conditional Expectation¶

\[E_Y[E[X|Y]] = E[X]\]

Example

Let \(X \sim Geometric(p)\).

Reminder

\(X\) is number of trials until first success in a sequence of i.i.d. (independent, identically distributed) \(Bernoulli(p)\) trials.

\[P(X=k) = (1-p)^{k-1}p\]

and the expectation of X:

\[E[X] = \sum_{k=1}^\infty k(1-p)^{k-1}p = ... = \frac{1}{p}\]

We can find this answer also by conditioning on a new random variable, \(Y\).

Let:

\[\begin{split}\displaystyle Y = \cases { 1 &\text{if first trial is success}\cr 0 &\text{if first trial is failure} }\end{split}\]

We have that \(P(Y=1)=p\) and \(P(Y=0)=1-p\).

\[E[X|Y=1] = 1\]\[E[X|Y=0] = 1 + E[X]\]

So,

\[E[X] = p*1 + (1+E[x])(1-p)\]\[E[X] = \frac{1}{p}\]

as before.

Special Case¶

If

\[\begin{split}\displaystyle X=I_A:= \cases { 1 &\text{if A occurs}\cr 0 &\text{if A does not occur} }\end{split}\]

Then \(E[X]=P(A)\).

So, \(P(A) = \sum_Y P(A|Y=y)P(Y=y)\).

This is dubbed “The Law of Total Probability”, and is a corollary of The Law of Total Expectation.

Example

\[X_0 = 0, X_n = \sum_{k=1}^n Y_k\]

where

\[Y_1, Y_2, ... , Y_n\]

are independent, with

\[P(Y_i = 1) = p\]

and

\[P(Y_i = -1) = 1-p =: q\]

What is the probability that \(X_n\) ever reaches state 1?

Let \(p\) denote this probability.

Similarly, what is \(p_r\)? where \(p_r\) is the probability that the walk ever reaches state \(r\), with \(r>1\).

Let:

\[\begin{split}\displaystyle Y = \cases { 1 &\text{if the walk ever reaches state 1}\cr 0 &\text{if not.} }\end{split}\]

Let \(A\) be the event that the walk ever reaches state \(r\).

Then,

\[P(A) = P(A|Y=1)P(Y=1) + P(A|Y=0)P(Y=0)\]

The condition \(A|Y=0\) is obviously self-contradictory given the definitions of \(A\) and \(Y\), so only the first term remains:

\[P(A) = P(A|Y=1)P(Y=1) = p_1 P(A|Y=1)\]\[= p_1*p_{r-1} = p_1^r\]

Let us now compute \(p_1\).

Let \(Y\) be the state after the first step;

\[\begin{split}\displaystyle Y = \cases { 1 &\text{with probability }p\cr -1 &\text{with probability }q }\end{split}\]
\[p_1 = P(\text{ever reaching }1 | Y=1)p + P(\text{ever reaching }1 | Y=-1)\]\[= p + P(\text{ever reaching state 2 from state 0})q\]\[= p + p_1^2q\]

Equivalently,

\[qp_1^2 + p_1 + p = 0\]

Solving this quadratic, we get

\[p_1 = \frac{-1 \pm \sqrt{1-4qp}}{2q}\]\[= \frac{1 \pm \sqrt{1-4q(1-q)}}{2q}\]\[= \frac{1 \pm \sqrt{4q^2-4q+1}}{2q}\]\[= \frac{1 \pm \sqrt{(-2q + 1)}}{2q}\]\[= \frac{1 \pm |1 - 2q|}{2q}\]

If \(q=\frac{1}{2}\), the roots are both 1, so \(p_1=1\)

If \(q<\frac{1}{2}\), then \(|1-2q| = 1 - 2q\), so roots are:

\[\frac{1 \pm (1-2q)}{2q} = 1 \text{ or } \frac{p}{q}\]

Only one of those (\(1\)) is a valid probability.

If \(q > \frac{1}{2}\), then \(|1-2q| = 2q-1\), so:

\[\frac{1 \pm (2q - 1)}{2q} = 1 \text{ or } \frac{p}{q}\]

Since the function of \(q\) ought to be continuous (not proven here), \(\frac{p}{q}\) is the correct root in this case.

Remark

The law of total expectation can also be used to compute a conditional expectation:

\[E[X|Y=y] = \sum_Z E[X|Y=y,Z=z]*P(Z=z|Y=y)\]
