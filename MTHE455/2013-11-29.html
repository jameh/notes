Example (Q1: Branching Process)

Recall

The extinction probability is the smallest solution to \(η=G(η)\), where \(G\) is the family size generating function.

Suppose family size is \(Geometric(p)\) on \(\{0,1,2,...\}\):

\[P(\text{family size is }k)=pq^k\]

where \(q=1-p\), for \(k=0,1,2,...\)

Suppose each child born survives with probability \(r\), and does not survive to the next generation with probability \(1-r\)

Let \(X\) be the number of offspring of any given individual that make it to next generation.

\[P(X=k)=\sum_{n=k}^\infty P(X=k|Y=n)pq^n\]

where \(Y\sim Geometric(p)\)

\[P(X=k)=\sum_{n=k}^\infty {n choose k}r^k(1-r)^{n-k}pq^n\]

For the extinction probability, we want the generating function of \(X\):

\[\begin{split}G_X(s)&=\sum_{k=0}^\infty s^kP(X=k)\\ &=\sum_{k=0}^\infty s^k\sum_{n=k}^\infty {n choose k}r^k(1-r)^{n-k}pq^n\\ &=\sum_{n=0}^\infty pq^n \sum_{k=0}^n {n choose k}(sr)^k(1-r)^{n-k}\\ &=\sum_{n=0}^\infty pq^n (sr+1-r)^n\\ &=p\sum_{n=0}^\infty (q(sr+1-r))^n\\ &=\frac{p}{1-q(sr+1-r)}\end{split}\]

Extinction probability satisfies:

\[η=\frac{p}{1-q(ηr+1-r)}\]

or

\[\begin{split}η(1-q(ηr+1-r))=p\\ qrη^2 -(1-q(1-r))η+p=0\end{split}\]

Example (2012 Final #4(a))

Firefly alternates between illuminated and not illuminated. At time \(t\), the firefly is illuminated if and only if the most recent event before time \(t\) in a Poisson process with rate \(λ\) occured in \([t-1,t]\).

At time \(t\), what is the probability that the firefly is illuminated?

Given \(N(t)=n\), the most recent event time is distributed as \(max(U_1,...,U_n)\), where the \(U_i\) are independent \(Unif(0,t)\) random variables.

So if \(A_t\) is the event that the firefly is illuminated at time \(t\) and \(L(t)\) is the time of the most recent event before time \(t\), then

\[\begin{split}P(A_t)&=\sum_{n=1}^\infty P(A_t | N(t)=n)P(N(t)=n)\\ &=\sum_{n=1}^\infty P(L(t)\in [t-1,t]|N(t)=n)P(N(t)=n)\end{split}\]
\[\begin{split}P(L(t)in [t-1,t]|N(t)=n) &= P(max(U_1,...,U_n)\geq t-1)\\ &=1-P(max(U_1,...,U_n) < t-1)\\ &= 1-P(U_1<t-1,...,U_n<t-1)\\ &=1-\left(\frac{t-1}{t}\right)^n\end{split}\]

So now,

\[P(A_t)=\sum_{n=1}^\infty \left(1-\left(\frac{t-1}{t}\right)^n\right)\frac{(λt)^n}{n!}e^{-λt}\]

... Taylor series expansion.

Example (Final Exam 2012 Problem 5(a))

If a discrete-time Markov chain is time-reversible, then show that the period of any state cannot be greater than 2.

Suppose \(p_{ij}>0\) for some \(j\). Then \(p_{ji}>0\) by time reversibility.

Then \(p_{ii}(2)\geq p_{ij}p{ji}>0\). Thus, the period must divide 2. QED
