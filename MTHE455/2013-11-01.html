Poisson Process¶

Note

See Takahara’s online notes Lecture 21 for info on the Exponential Distribution

Let \(\{X_t: t\geq 0\}\) be a stochastic process with index set \([0,\infty)\)

Definition

An increment is of the form

\[X(t)-X(s), s\geq t\]

Definition

The process \(\{X_t: t\geq 0\}\) is said to have stationary increments if the distribution of \(X_t-X_s\) depends only on the difference \(t-s\).

Definition

The process \(\{X(t): t\geq 0\}\) is said to have independent increments if any two increments involving disjoint intervals are independent.

Example

If \(Z_i\), \(i=1,2,...\) are independent and identically distributed, and \(X_n=\sum_{i=1}^nZ_i\), then \(\{X_n:n\geq 0\}\) has both stationary and independent increments.

Definition

Poisson Process

A continuous-time stochastic process

\[\{N(t): t\geq 0\}\]

is a Poisson process with rate \(λ\) if

  1. \(N(0)=0\)
  2. It has stationary and independent increments
  3. The distribution of \(N(t)\) is Poisson with mean \(λt\)
\[\begin{split}P(N(t)=k)=\frac{(λt)^k}{k!}e^{-λt}, && k=0,1,2\end{split}\]

From this definition

  • For \(s<t\), the distribution of \(N(t)-N(s)\) is the same as the distribution of \(N(t-s)-N(0)\), by stationary increments, and since \(N(0)=0\), we have

    \[N(t)-N(s)\sim Poisson(λ(t-s))\]

  • The process is non-decreasing since

    \[N(t)-N(s)\sim Poisson(λ(t-s))\geq 0\]

    with probability 1

  • The sample space of the Poisson process is \(S=\{0,1,2,...\}\)

Landau’s “Little oh” Notation¶

The notation \(o(h)\) is called “little oh of h”, and does not refer to any specific function

It specifies any function of \(h\) that goes to 0 at a faster rate than \(h\), as \(h\to 0\).

In other words, \(\frac{o(h)}{h}\to 0\) as \(h\to 0\).

We get things like

\[\begin{split}o(h)+o(h)&=o(h)\\ o(h)o(h)&=o(h)\\ c\cdot o(h)&=o(h)\end{split}\]

where \(c\) is any constant

Example

\(h^k\) is \(o(h)\) for any \(k>1\)

\[\begin{split}\sum_{k=2}^\infty c_k h^k\text{ with }|c_k|<1 && \forall k\end{split}\]

is \(o(h)\)

Definition

Poisson Process (def 2)

A continuous time process \(\{N(t):t\geq 0\}\) is a Poisson process with rate \(λ>0\) if

  1. \(N(0)=0\)
  2. It has stationary and independent increments
  3. The probabilities:
\[\begin{split}P(N(h)=1)&=λh+o(h)\\ P(N(h)\geq 2)&=o(h)\\ P(N(h)=0)&=1-λh+o(h)\end{split}\]

Let us show that the first definition implies this definition.

First, note that \(N(h)\sim Poisson(λh)\)

So

\[\begin{split}P(N(h)=0)&=e^{-λh}\\ &=1-λh+\frac{(λh)^2}{2}-\frac{(λh)^3}{3!}+...\\ &=1-λh+o(h)\end{split}\]

Also,

\[\begin{split}P(N(h)=1)&=λhe^{-λh}\\ &=λh-λ^2h^2+\frac{(λh)^3}{2}-\frac{(λh)^3}{3!}+...\\ &=λh+o(h)\end{split}\]

By subtraction,

\[\begin{split}P(N\geq 2)&=1-(λh+o(h))-(1-λh+o(h))\\ &=o(h)\end{split}\]
