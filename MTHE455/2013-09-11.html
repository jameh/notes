Time Homogeneity¶

Definition

A stochastic process is time-homogeneous if:

\[P(X_n = j | X_m = i) = P(X_{n+k} = j | X_{m+k} = i)\]

for all \(i\), \(j \in S\) and for all \(m \leq n\).

The simple symmetric random walk (from last lecture) is time-homogeneous.

Definition

A stochastic process is stationary if:

\(P(X_{t_1} = i_1, ..., X_{t_k} = i_k) = P(X_{t_1}+s = i_1, ..., X_{t_k}+s = i_k)\)

for all times \(t_1, ..., t_k\) and any \(k \geq 1\) and any states \(i_1, ... i_k\)

Example

Galton-Watson Branching Process

We assume that the family size distribution on \(\{0,1,2,3,...\}\) is the same for all individuals and all family sizes are independent.

Let \(Y_{n,k}\) be i.i.d random variable, each with family size distribution \(n=0,1,2,...\)

Let \(X_n\) be the number of individuals in generation \(n\).

\[X_0 = 1\]\[X_1 = Y_{0,1}\]\[X_2 = Y_{1,1} + Y_{1,2}\]\[...\]\[X_n = Y_{n-1, 1} + Y_{n-1, 2} + ... + Y_{n-1, X_{n-1}}\]

This process is time-homogeneous, but not stationary.

Definition

A stochastic process is spatially homogeneous if

\[P(X_1 = i_1, ..., X_n = i_n) = P(X_1 + k = i_1, ..., X_n + k = i_n)\]

Conditional Expectation (a review)¶

If \(X\) and \(Y\) are discrete random variables, then the expectation of \(X\) given \(Y=y\) is:

\[E[X|Y=y] = \sum_\chi x P(X=x | Y=y)\]

If \(X\) and \(Y\) are continuous,

\[E[X | Y=y] = \int\limits_{-\infty}^\infty x f(x|y)dx\]

where \(f(x|y)\) is the conditional pdf of \(X\) given \(Y=y\). (\(f(x|y) = \frac{f(x,y)}{f_Y(y)}\))

When we write \(E[X|Y]\), we think of it as a function of \(Y\) whose value at \(y\) is \(E[X|Y=y]\).

So we can take the expected value of it as:

\[\sum\limits_Y E[X|Y=y]P(Y=y)\]\[= \sum\limits_Y\sum\limits_X xP(X=x | Y=y)P(Y=y)\]\[= \sum\limits_Y\sum\limits_X x \frac{P(X=x, Y=y)}{P(Y=y)}P(Y=y)\]\[= \sum\limits_Y\sum\limits_X x P(X=x, Y=y)\]\[= \sum\limits_X xP(X=x) = E[X]\]
