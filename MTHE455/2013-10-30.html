Example of Markov chain Monte Carlo and Time Reversibility¶

Example using Markov chain Monte Carlo¶

Bayesian Statistical Inference¶

In statistics, we observe data \(x_1, ... ,x_n\) and a common statistical model for the data is that realizations of the variables \(X_1, ... ,X_n\), where it is often assumed that the \(X_i\) are independent and identically distributed each with an unknown pdf/pmf \(f(x|θ)\), where \(θ\) is an unknown parameter which indexes a family of possible distributions for the \(X_i\).

The problem is to infer the value of \(θ\) that corresponds to the observed data.

Suppose that \(θ\in Θ\), where \(Θ\) is the parameter space. In general, \(Θ\) is a continuous space, but let us assume \(Θ\) is discrete here, but \(Θ\) could be multidimensional.

In the Bayesian approach, a distribution \(Π(θ)\) is placed on \(Θ\), which is called prior distribution on \(Θ\).

With the two ingredients \(f(x|θ)\) and \(Π(θ)\), we can compute (in principle) the conditional distributions of \(θ\) given the data \(x\), using Baye’s theorem:

\[Π(θ|x)=\frac{Π(θ)f(x|θ)}{\sum_{θ'\in Θ}Π(θ')f(x|θ')}\]

Note

The denominator (normalization constant) can often be not feasible to compute. But for Markov chain Monte Carlo, the denominator need not be computed (it cancels in the ratio \(Π(θ_1|x)/Π(θ_2|x)\))

Time Reversibility¶

Let \(\{X_n: n\geq 0\}\) be a Markov chain. Suppose it is stationary with stationary distribution \(Π\)

Extend the time index to negative values, so we have \(\{X_n: -\infty<n<\infty\}\)

Let \(Y_n=X_{-n}\), for \(n\in \mathbb Z\)/

The process \(\{Y_n; n\in \mathbb Z\}\) is called the reversed process.

The \(Y\) process is again a Markov chain.

We have that given \(X_n=i\), \(X_m\) is independent of \(X_k\) for any \(m>n\) and any \(k<n\).

\((X_k)\) ... \((X_n=i)\) ... \((X_m)\) tells us that \(X_m\) is independent of \(X_k\), and, reversing time:

\((X_{-m})\) ... \((X_n={-i})\) ... \((X_{-k})\) we have (by symmetry of independence) that \(X_k\) is still independent of \(X_m\), for any \(-k>-n\) and any \(-m<-n\).

Moreover, the reversed chain has the same stationary distribution, since the long run proportion of time the process spends in state \(i\) is the same whether going forwards or backwards in time.

For local flow between a pair of states \(i\) and \(j\), however, the flow going forwards in time need not be the same as the flow going backwards in time.

Let us compute the transition probabilities of the reversed chain.

\[\begin{split}P(Y_n=j|Y_{n-1}=i)&=P(X_{-n}=j|X_{-(n-1)}=i)\\ &=\frac{P(X_{-n}=j, X_{-(n-1)}=i)}{P(X_{-(n-1)}=i)}\\ &=\frac{P(X_{-(n-1)}|X_{-n}=j)P(X_{-n}=j)}{P({X_{-(n-1)}=i})}\\ &=\frac{p_{ji}Π_j}{Π_i}\end{split}\]

The reversed chain will have the same transition probabilities, \(\{p_{ij}\}\), as the forward chain if and only if

\[\begin{split}p_{ij}=\frac{p_{ji}Π_j}{Π_i}\\ \iff Π_ip_{ij}=Π_jp_{ji}\end{split}\]

if and only if local balance is satisfied.

In this case, we say that the \(X\) Markov chain is time-reversible.
