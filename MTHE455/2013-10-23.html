Set \(Π^{(0)}\) to be some arbitrary distribution over \(S\). Set \(Π^{(1)}=Π^{(0)}P\), and \(Π^{(k)}=Π^{(k-1)}P\).

Then \(Π^{(k)}\to Π\) since

\[\begin{split}Π^{(k)}&=Π^{(k-1)}P\\ &=Π^{(k-2)}P^2\\ &...\\ &=Π^{(0)}P^k\end{split}\]

For \(k\) large, \(p_{ij}(k)\) will be close to \(Π_j\).

Then \(Π_j^{(k)}=\sum_{i\in S}Π_i^{(0)}p_{ij}(k)\approx\sum_{i\in S}Π_i^{(0)}Π_j=Π_j\)

Practical note

In many practical modelling situations, the number of nonzero entries in any given row of \(P\) is bounded by some \(M\), where \(M\) is much smaller than the total number of states \(|S|\). Most of the entries in \(P\) are zero. So one only needs to store the nonzero entries of \(P\) and where they are in \(P\) to compute \(Π^{(k)}=Π^{(k-1)}P\), which can make the complexity and storage requirements linear in \(|S|\), and so if we’re lucky we can find \(Π\) for very large state spaces e.g. hundreds of thousands of states.

Flow¶

We have the following interpretations:

  • \(Π_i\) is the long run proportion of time the chain is in state \(i\)
  • \(p_{ij}\) is the long run proportion of time that the chain transitions from state \(i\) to \(j\)
  • \(Π_ip_{ij}\) is the long run proportion of all transitions that are from \(i\) to \(j\). prop

We call this the flow from state \(i\) to state \(j\).

Summing this flow over all \(i\in S\), we get the flow into state \(j\).

Interchanging \(i\) and \(j\): \(Π_jp_{ji}\) is the flow from \(j\) to \(i\)

Summing this over \(i\) is the flow out of state \(j\).

Note

The stationary distribution satisfies:

\[Π_j=\sum_{i\in S}Π_ip_{ij}\]

which is the same as:

(1)\[Π_j=\sum_{i\in S}Π_jp_{ji}=\sum_{i\in S}Π_ip_{ij}\]

Thus, the stationary probabilities make the flow into \(j\) equal to the flow out of \(j\).

For this reason, the equations \(Π_j=\sum_{i\in S}Π_ip_{ij}\) are called the balance equations, or global balance equations.

In (1), if each of the terms is equal (for each \(i\)) then we say that local balance or detailed balance is satisfied.

Clearly, if \(Π\) satisfies local balance, that is \(Π_jp_{ji}=Π_ip_{ij}\) for all \(i,j\in S\) then global balance will be satisfied and so \(Π\) is the stationary distribution.

The converse is not true: one can easily have a Markov chain where \(p_{ij}>0\) but \(p_{ij}=0\) in which case, \(Π_ip_{ij}\neq Π_jp_{ji}=0\).

There are \(\left(\matrix{|S| \\ 2}\right)\) (nontrivial) local balance equations. However, often, the majority of these equations are for states \(i,j\) such that \(p_{ij}=p_{ji}=0\)

Moreover, each local balance equation involves only 2 unknowns.

Example

(4.36 in text)

Consider a graph with \(N\) nodes and arcs (edges) between nodes and the arc between node \(i\) and node \(j\) (if there is an arc) has weight \(w_{ij}>0\). We assume \(w_{ji}=w_{ij}\). A particle performs a random walk on the graph. When at node \(i\), the particle will move next to node \(j\) (if there is an arc) with probability \(\frac{w_{ij}}{\sum_{k\in N_i}w_{ik}}\) (where \(N_i\) is the set of all nodes joined by an arc to node \(i\)).

Let \(X_n=\) node label at step \(n\), \(n=0,1,2,...\)

The transition probabilities are given by \(p_{ij}=\frac{w_{ij}}{\sum_{k\in N_i}w_{ik}}\).

Assume graph is connected, so the Markov chain is irreducible.

In this situation, we can try to solve the local balance equations (since \(p_{ij}=p_{ji}\) for all \(i,j\)). For 2 states \(i\) and \(j\) connected by an arc,

\[\begin{split}Π_i\frac{w_{ij}}{\sum_{k\in N_i}w_{ik}}&=Π_j\frac{w_{ji}}{\sum_{l\in N_j}w_{jl}}\\ \frac{Π_i}{\sum_{k\in N_i}w_{ik}}&=\frac{Π_j}{\sum_{l\in N_j}w_{jl}}\end{split}\]

So setting \(Π_i=k\sum_{k\in N_i}w_{ik}\), the local balance equations will be satisfied.

So

\[Π_i=\frac{\sum_{k\in N_i}w_{ik}}{\sum_{i\in S}\sum_{k\in N_i}w_{ik}}\]
