Introduction to Stochastic Process¶

Definition

A Stochastic Process is a collection of random variables,

\[\left\{ X_{u}; u \in I \right\}\]

There are 2 fundamental properties of a stochastic process.

  1. The index set \(I\). Typically, we will take \(I=\left\{ 0, 1, 2, ... \right\}\) or \(I=[0,\infty)\) In the first case, \(I\) is discrete, in the second, \(I\) is a continuous index set.
  2. If \(S_{u}\) is the sample space of \(X_{u}\), then \(S = \cup_{u \in I} S_{u}\) is the state space of the stochastic process. (i.e. \(S\) is the set of all possible values of \(X_{u}'s\))

Example¶

Let \(Y_{1}, Y_{2}, ..., Y_{n}\) be independent random variables with:

\[P(Y_{i} = -1) = P(Y_{i} = 1) = \frac{1}{2}, \forall i \in \left\{1, 2, ..., n\right\}\]

Let \(X_{0}=0\) and \(X_{n} = \sum_{i=1}^{n} Y_{i}\) for \(n \ge 1\)

Here, \(I=\mathbb Z_{\ge 0}\) and \(S=\mathbb Z\)

For every given \(n\), \(X_{n}\) can take values \(-n, -n+2, ..., n-2, n\)

The Distribution of Stochastic Processes¶

Recall, a random variable \(X\), is a function from \(\Omega \to \mathbb R\) where \((\Omega, P)\) is a probability space (\(P\) is the probability measure on \(\Omega\))

The distribution of \(X\) is the probability law on \(\Bbb R\) by \(P\) under the mapping \(X\):

i.e. for \(A \subset \Bbb R\),

\[P(X \in A) = P(X^{-1}A)\]

where

\[X^{-1}A = \left\{\omega \in \Omega | X(\omega) \in A \right\}\]

For a stochastic process with index set \(I\), the collection \(\left\{X_{u} | u \in I \right\}\) can be thought of as a mapping from \(\Omega \to \Bbb R^{I}\), where \(\Bbb R^{I}\) is the set of all functions from \(I\) to \(\Bbb R\).
