Generating Function ExampleÂ¶

Recall,

\[G_r(s) = \sum_{n=0}^\infty f_r(n)s^n\]

Where \(f_r(n)=P(T_r=n|X_0=0)\)

\[\begin{split}T_r=\begin{cases} \text{first time walk reaches }r & r\geq 1 \\ \text{first time walk returns to 0,}& r=0 \end{cases}\end{split}\]
\[G_1(s) = \frac{1-\sqrt{1-4pqs^2}}{2qs}\]

Now consider \(G_0(s)\). We have (conditioning on \(X_1\)):

\[f_0(n)=P(T_0=n|X_0=0)\]\[=P(T_0=n|X_1=1,X_0=0)P(X_1=1|X_0=0)+P(T_0=n|X_1=-1,X_0=0)P(X_1=-1|X_0=0)\]\[=P(T_0=n|X_1=1)p + P(T_0=n|X_1=-1)q\]

Consider \(P(T_0=n|X_1=-1)\)

By temporal and spatial homogeneity,

\[P(T_0=n|X_1=-1)=f_1(n-1)\]

For \(P(T_0=n|X_1=1)\), the picture is:

By flipping the picture around the \(x\) - axis, interchange roles of \(p\) and \(q\), and using temporal spatial homogeneity,

\[P(T_0=n|X_1=1)f_1^*(n-1)\]

where “*” refers to a random walk where the “up” probability is \(q\), and the “down” probability is \(p\).

So,

\[f_0(n)=f_1^*(n-1)p+f_1(n-1)q\]

Multiply by \(s^n, n\geq 1\) and sum over \(n\):

\[\sum_{n=1}^\infty f_0(n)s^n=\sum_{n=1}^\infty pf_1^*(n-1)s^n+\sum_{n=1}^\infty qf_1(n-1)s^n\]

or

\[G_0(s)=psG_1^*(s)+qsG_1(s)\]\[=ps\frac{1-\sqrt{1-4pqs^2}}{2ps}+qs\frac{1-\sqrt{1-4qps^2}}{2qs}\]\[=1-\sqrt{1-4pqs^2}\]

Now, we can compute the probability:

\[P(\text{walk ever returns to }0|X_0=0)\]\[=G_0(1)\]\[=1-\sqrt{1-4pq}\]\[=1-\sqrt{1-4(1-q)q}\]\[=1-\sqrt{1-4+4q^2}\]\[=1-\sqrt{(1-2q)^2}\]\[=1-|1-2q|\]

If \(q=\frac{1}{2}\), then \(G_0(1)=1\)

If \(q<\frac{1}{2}\), then \(G_0(1)=2q<1\)

If \(q>\frac{1}{2}\), then \(G_0(1)=2p<1\)

Therefore, if \(p=\frac{1}{2}\), then state 0 and, by irreducibility, all states, are recurrent.

If \(p\neq \frac{1}{2}\), then state 0 (and all states) are transient.

Note

Note that when \(p=\frac{1}{2}\), \(T_0\) is a proper random variable; i.e. it is real-valued with probability 1.

If \(p\neq \frac{1}{2}\), then the probability that \(T_0\) is real-valued is less than 1 (i.e. \(P(T_0=\infty|X_0=0)>0\)). In this case, we call \(T_0\) a defective random variable.

When \(p=\frac{1}{2}\), we can use \(G_0(s)\) to find \(E[T_0]\), via:

\[E[T_0]=G_0'(1)\]\[G_0'(s)=\frac{8pqs}{\sqrt{1-4pqs^2}}\]\[=\frac{2s}{\sqrt{1-s^2}}\]

when \(p=q=\frac{1}{2}\).

So, \(G_0'(1)=\infty\) (i.e. the expected time to return to 0 is \(\infty\))

Definition

A recurrent state is called null recurrent if the expected time to return is \(+\infty\). Otherwise, it is called positive recurrent.

Theorem

Recurrent classes are closed.

That is, one the Markov chain is in a recurrent class, it will never leave the class (with probability 1).

Proof:

Suppose the recurrent class was not closed.

Then there must be some state \(i\) in the class and some state \(j\) outside the class such that \(p_{ij}>0\). But it must be possible to return to state \(i\) with positive probability since \(i\) is recurrent. But then, \(i\) and \(j\) communicate; so \(j\) could not have been outside the class.

Theorem

Let \(X_i, i=1,2,...\) be i.i.d. random variable and \(N\) be a random variable independent of \(X_i`s on the non-negative integers, with generating functions :math:`G_X(s)\) and \(G_N(s)\) resp.

Let \(Y=X_1+X_2+...X_n\)

Then the generating function of \(Y\) is:

\[G_Y(s)=G_N(G_X(S))\]

Proof:

We have

\[G_Y(s) = E[s^Y]\]\[=E[s^{X_1+...+X_N}]\]\[=\sum_{n=0}^{\infty}E[s^{X_1+...+X_N} |N=n]P(N=n)\]\[=\sum_{n=0}^\infty E[s^{X_1+...+X^n}]P(N=n)\]\[=\sum_{n=0}^\infty E[\prod_{i=1}^n s^{X_i}]P(X=n)\]\[=\sum_{n=0}^\infty \prod_{i=1}^n E[s^{X_i}]P(X=n)\]\[=\sum_{n=0}^\infty \prod_{i=1}^n G_X(s)^n P(X=n)\]\[=E[G_X(s)^N]\]\[=G_N(G_X(s))\]
