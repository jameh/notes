Recall the global balance equations

\[ΠG=0\]
\[v_jΠ_j=\sum_{i\in S}Π_iq_{ij}\]

“flow out”=”flow in”

Local balance would say that flow from \(i\) to \(j\) equals flow from \(j\) to \(i\), for any pair of states \(i\neq j\), i.e.

\[Π_iq_{ij}=Π_jq_{ji}\]

There are \({|S| choose 2}\) equations (if \(|S|\) is finite).

Note

These equations will not be satisfied if \(q_{ij}>0\) and \(q_{ji}=0\) (or \(q_{ij}=0\) and \(q_{ji}>0\)) for some \(i\) and \(j\)

If \(Π\) satisfies the local balance equations, then

\[\begin{split}\sum_{i\in S}Π_iq_{ij}&=\sum_{i\in S}Π_jq_{ji}\\ &=Π_j\sum_{i\in S}q_{ji}\\ &=Π_jv_j\end{split}\]

which is the global balance equation for state \(j\).

Example (Birth-Death Processes)

Suppose \(S=\{0,1,2,...\}\)

\[\begin{split}q_{i,i+1}=λ_i>0\end{split}\]

for \(i\geq 0\).

\[\begin{split}q_{i,i-1}=μ_i>0\end{split}\]

for \(i\geq 1\).

The \(λ_i\) are called the birth rates, the \(μ_i\) are called the death rates.

The local balance equations are

\[Π_iq_{i,i+1}=Π_{i+1}q_{i+1,i}\]

\(i\geq 0\). Or

\[Π_iλ_i=Π_{i+1}μ_{i+1}\]

So

\[\begin{split}Π_{i+1}&=\frac{λ_i}{μ_{i+1}}Π_i\\ &=\frac{λ_i}{μ_{i+1}}\frac{λ_{i-1}}{μ_i}Π_{i-1}\\ &=\frac{λ_i...λ_0}{μ_{i+1}...μ_1}Π_0\end{split}\]

For a stationary distribution to exist, we must have

\[\begin{split}Π_0(1+\frac{λ_0}{μ_1}+...+\frac{λ_0...λ_{i-1}}{μ_1...μ_i}+...)}<\infty\end{split}\]

i.e.

\[1+\sum_{i=1}^\infty \frac{λ_0...λ_{i-1}}{μ_1...μ_i}+...}\]

in which case

\[Π_i=\frac{λ_0...λ_{i-1}}{μ_1...μ_i}(1+\sum_{i=1}^\infty \frac{λ_0...λ_{i-1}}{μ_1...μ_i})^{-1}\]

Special Case: M/M/1 Queue

  • A single server first-in-first-out queue with arrivals coming as a Poisson process with rate \(λ\) and service times independent \(Exponential(μ)\)

Let \(X(t)\) be the # of customers in the system at time \(t\).

Then \(\{X(t):t\geq 0\}\) is a birth-death process with \(λ_i=λ\) for all \(i\) and \(μ_i=μ\) for all \(i\). A stationary distribution exists if

\[\begin{split}1+\frac{λ}{μ}+(\frac{λ}{μ})^2+...<\infty\end{split}\]

which is true iff \(λ<μ\). In this case,

\[Π_i=(\frac{λ}{μ})^i(1-\frac{λ}{μ})\]

for \(i=0,1,2,3,..\)

i.e. \(Π\) is \(Geometric(\frac{λ}{μ})\).

Example

Consider a system with \(r\) servers.

Service times at server \(i\) are \(Exponential(μ_i)\).

A total of \(N\) customers move among these servers. When a customer finishes service at a server, it chooses a different server at random and moves to that server.

Let \(X(t)=(n_1(t),...,n_r(t)\) where \(n_r(t)\) is the # of customers at server \(i\) at time \(t\).

Find the stationary distribution.

First, the state space is

\[S=\{(n_1,...,n_r):n_i\text{ is a non-negaative integer, }\sum_{i=1}^r n_i=N\}\]

From state \(n=(n_1,...,n_r)\), the process can only move to the state of the form

\[n-e_i+e_j\]

where \(e_i\) is vector of all zeros except with a 1 in position \(i\).

Then what is \(q_{n,n-e_i+e_j}\)?

This is

\[\begin{split}\sum_{k=1}^r μ_kI_{\{n_k>0\}}\left(\frac{μ_iI_{\{n_i>0\}}}{\sum_{k=1}^r μ_kI_{\{n_k>0\}}}\right)\frac{1}{r-1}\\ =\frac{μ_i}{r-1}\end{split}\]

(assuming \(n_i>0\)).

Similarly, \(q_{n-e_i+e_j,n}=\frac{μ_i}{r-1}\)

Thus the local balance equations:

\[Π_n\frac{μ_i}{r-1}=Π_{n-e_i+e_j}\frac{μ_j}{r-1}\]

These are shrewdly solved by letting

\[Π_n=C\frac{1}{μ_1^{n_1}...μ_r^{n_r}}\]
