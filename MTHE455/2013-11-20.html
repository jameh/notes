More Continuous-Time Markov ChainsÂ¶

Last time we described a class of continuous-time processes.

Further Remarks

  1. By memorylessness of the exponential distribution, if we observe \(X(t)=i\) for an arbitrary time \(t\), the remaining times until alarm of the timers that started when the process entered state \(i\) are still independent \(Exponential(q_{ij}),j\in S_i\)

    Recall

    \(S_i\) is the set of states \(j\) that we can get to from \(i\).

    And the remaining holding time in state \(i\) is still \(Exponential(v_i)\). Where

    \[v_i = \sum_{j\in S_i}q_{ij}\]

  2. These processes have the Markov property:

    \[\begin{split}P(X(t)=j|X(s)=i,X(t_{n-1})=i_{n-1},...,X(t_1)=i_1)\\ =P(X(t)=j|X(s)=i)\end{split}\]

    for \(0\leq t_1\leq t_2\leq ...\leq t_{n-1}\leq s\leq t\)

    because in both the LHS and RHS, the probability of the event \(\{X(t)=j\}\) depends only on the evolution of the process from time \(s\), and given \(X(s)=i\), any information about the process before time \(s\) does not change the probabilistic properties of that evolution (because of the memorylessness)

  3. These processes are time-homogeneous:

    \[P(X(t)=j|X(s)=i)=P(X(t-s)=j|X(0)=i)\]

    because for any time \(u\), given \(X(u)=i\), the probabilistic evolution from time \(u\) is the same for any \(u\).

  4. By (2) and (3), these processes are time-homogeneous, continuous-time Markov chains. In this course, any reference to a “continuous-time Markov chain” will mean a process of this class.

  5. To specify this process, it’s not necessary to first specify the \(p_{ij}\) and \(v_i\) then define

    \[q_{ij}=v_ip_{ij}\]

One can simply specify the \(q_{ij}\) (with \(q_{ii}=0\)) from the start.

We also assume that \(q_{ij}>0\) for only finitely-many \(j\), for each \(i\).

Then

\[\begin{split}v_i = \sum_{j: q_{ij}>0}q_{ij}\end{split}\]

is the parameter of the exponential holding time in state \(i\).

As well,

\[\begin{split}p_{ij}=\frac{q_{ij}}{\sum_{k:q_{ik}>0}q_{ik}}\end{split}\]

is the probability of going to state \(j\) when the process leaves state \(i\), and \(p_{ij}>0\) if \(q_{ij}>0\) and \(p_{ij}=0\) if \(q_{ij}=0\).

So \(S_i\) is the set of states \(j\) that the process can jump to upon leaving state \(i\) is:

\[\begin{split}\{j: q_{ij}>0\}\end{split}\]

\(q_{ij}\) is called the rate of going from \(i\) to \(j\).

These parameters are usually put into a matrix called the generator matrix, denoted by \(G\), where

\[\begin{split}g_{ij}= \begin{cases} q_{ij} &\text{ if }i\neq j\\ -v_i&\text{ if }i=j \end{cases}\end{split}\]

Let us now consider what such a process does in an interval of length \(h\).

  1. First,

    \[\begin{split}P(0\text{ transitions by time }h|X(0)=i)\\ =P(T_i>h)\end{split}\]

    where \(T_i\) is the holding time in state \(i\).

    \[\begin{split}=e^{-v_ih}\\ =1-{v_ih}+\frac{(v_ih)^2}{2}-\frac{(v_ih)^3}{6}+...\\ =1-v_ih+o(h)\end{split}\]

  2. Next, consider

    \[\begin{split}P(\text{2 or more transitions by time }h|X(0)=i)\\ \sum_{j\neq i}P(\text{2 or more transitions by time }h|X(0)=i,\text{1st transition is to }j)P(\text{1st transition is to }j|X(0)=i)\\ =\sum_{j\neq i}P(T_i+T_j)p_{ij}\end{split}\]

    where \(T_j\) is the holding time in state \(j\).

    Note

    \[\{T_i+T_j\leq h}\subset\{T_i\leq h,T_j\leq h\}\]

    So,

    \[\begin{split}P(T_i+T_j\leq h)&\leq P(T_i\leq h,T_j\leq h)\\ &= P(T_i\leq h)P(T_j\leq h)\\ &= (1-e^{-v_ih})(1-e^{-v_jh})\\ &= (v_ih+o(h))(v_jh+o(h))\\ &= v_iv_jh^2 + o(h)\\ &= o(h)\end{split}\]

    So

    \[P(\text{2 or more transitions by time }h|X(0)=i)=o(h)\]

    By subtraction,

    \[\begin{split}P(\text{exactly 1 transition by time }h|X(0)=i)\\ =1-(o(h)+1-v_ih+o(h))\\ =v_ih+o(h)\end{split}\]
