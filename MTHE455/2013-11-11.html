More Poisson¶

If \(\{N(t):t\geq 0\}\) is a Poisson process with rate \(λ\) and each event is type 1 with probability \(p\) and type 2 with probability \(1-p\), then if

\(N_1(t)\) is the number of type 1 events in \([0.t)\) \(N_2(t)\) is the number of type 2 events in \([0,t)\)

Then \(\{N_1(t):t\geq 0\}\) and \(\{N_2:t\geq 0\}\) are Poisson processes with rates \(λp\) and \(λ(1-p)\) respectively.

Let us see that these 2 processes are independent. (i.e. \(P(N_1(t_1)=n_1,N2(t_2)=n_2) = P(N_1(t_1)=n_1)P(N_2(t_2)=n_2)\) for any \(t_1,t_2\) and \(n_1,n_2\)).

First, suppose \(t_1=t_2=t\)

Compute

\[\begin{split}P(N_1(t),N_2(t)=n_2)\\ &=P(N_1(t),n_1,N_2(t)=n_2|N(t)=n_1+n_2)P(N(t)=n_1+n_2)\\ &={n_1+n_2 choose n_1}p^{n_1}(1-p)^{n_2}\frac{(λt)^{n_1+n_2}}{(n_1+n_2)!}e^{-λt}\\ &=[\frac{(λpt)^{n_1}}{n_1!}e^{-λ[t]}][\frac{[λ(1-p)t]^{n_2}}{n_2!}e^{-λ(1-p)t}]\\ &=P(N_1(t)=n_1)P(N_2(t)=n_2)\end{split}\]

Now assume (wlog) \(t_1<t_2\).

\[\begin{split}P(N_1(t_1)=n_1,N_2(t_2)=n_2)\\ =\sum_{k=0}^{n_2}P(N_1(t_1)=n_1,N_2(t_2)=k,N(t_2)-N_2(t_1)=n_2-k)\end{split}\]

All three events in the summand proabbility are independent

\[\begin{split}=P(N_1(t_1)=n_1)\sum_{k=0}^{n_2}P(N_2(t_1)=k,N_2(t_2)-N_2(t_1)=n_2-k)\\ =P(N_1(t_1)=n_1)P(N_2(t_2)=n)2\end{split}\]

Example

Note

Problem 1 will be similar to this

Consider a particle that moves among the vertices of a square. At each step, it chooses one of its neighbouring vertices at random, and moves to that vertex. Find the expected number of steps, starting at a given vertex, for the particle to get to the opposite vertex.

Square:

1-2

3-4

Matrix:

\[\begin{split}P=\matrix{0 && 0.5 && 0.5 && 0 \\ 0.5 && 0 && 0 && 0.5 \\ 0.5 && 0 && 0 && 0.5 \\ 0 && 0.5 && 0.5 && 0}\end{split}\]

Let \(M_2\) be the expected number of steps to get to vertex 2 edges away.

Let \(M_1\) be the expected number of steps to get to a particular vertex that is one edge away

Condition on first move, we get

\[M_2 = \frac{1}{2}(1+M_1)+\frac{1}{2}(1+M_1)=1+M_1\]

Now find \(M_1\). Condition on first move again.

\[M_1 = \frac{1}{2}(1) + \frac{1}{2}(1+M_2)\]

Solving,

\[\begin{split}M_2 = 1+1+\frac{1}{2}M_2=1+\frac{1}{2}M_2\\ M_2 = 4\end{split}\]

Conditional Distribution of Arrival Times Given \(N(t)=n\)¶

Let \(S_1,...,S_n\) be the event times of the first \(n\) events.

Let

\[I_k=[t_k,t_k+h)\]

The joint conditional density of \(S_1,...,S_n\) at \((t_1,...,t_n)\) given \(N(t)=n\) can be written as

\[\lim_{h\to 0}\frac{P(S_k\in I_k,k=1,...,n|N(t)=n)}{h^n}\]

Consider

\[\begin{split}P&(S_k\in I_k,k=1,...,n | N(t)=n)\\ &=\frac{P(S_k\in I_k,k=1,...,n, N(t)=n)}{P(N(t)=n)}\\ &=\frac{P(\text{exactly 1 event in }I_k,k=1,...,n,\text{ 0 events in }J_k,k=1,...,n+1)}{P(N(t)=n)}\\ &=\frac{(λhe^{-λh})^ne^{-λ(t-nh)}}{\frac{(λt)^n}{n!}e^{-λt}}\\ &=\frac{h^nn!}{t^n}\end{split}\]

now divide by \(h^n\) and get

\[g(t_1,...,t_n|N(t)=n)=\frac{n!}{t^n}\]

for \(t_t<t_2<...<t_n\)

recall: order statistics
