Generating Functions¶

\[G(s) = \sum_{n=0}^\infty a_n s^n\]

for a sequence \(\{a_n\}_{n=0}^\infty\)

If the \(a_n\) are \(P(X=n)\) for some random variable \(X\) on \(\{0,1,2,...\}\), then \(G(s)\) is called a probability generating function.

Also, \(G'(1)=E[X]\) in this case.

Since \(G'(s) = a_1+2a_2s + ... + na_ns^{n-1}+...\)

So, \(G'(1) = a_1+2a_2 + ... + na_n+...=E[X]\)

The nth derivative of \(G(s)\) at \(s=0\) is

\[n!a_n\]

That is, \(G^{(n)}(0)=n!a_n\) or,

\[a_n = \frac{1}{n!}G^{(n)}(0)\]

So, \(G\) determines \(a_n\) for every \(n\). So there is a one-to-one correspondence between sequences and generating functions.

Simple Random Walk via Generating Functions¶

Let \(T_r=\) time the walk first reaches state \(r\), \(r \geq 1\).

\(T_0=\) time the walk first returns to 0.

Let \(f_r(n)=P(T_r=n|X_0=0)\) for \(r \geq 0\), \(n \geq 0\).

Note that \(f_r(0)=0\) for all \(r\geq 0\)

Let

\[G_r(s) = \sum_{n=0}^\infty f_r(n)s^n\]

be the generating function of the sequence \(\{f_r(0),f_r(1),f_r(2),...\}\) for \(r \geq 0\),

Note

\[G_1(1) = \text{ probability walk ever goes to state 1.}\]

and

\[G_0(1) = \text{ probability walk ever returns to state 0.}\]

First, consider \(G_r(s)\), \(r>1\). Consider

\[f_r(n) = P(T_r=n|X_0=0)\]\[= \sum_{k=0}^\infty P(T_r=n|T_1=k,X_0=0)P(T_1=k|X_0=0)\]\[= \sum_{k=0}^n P(T_r=n|T_1=k) f_1(k)\]

because of Markov property, def’n of \(f_1(k)\)

\[= \sum_{k=0}^n f_{r-1}(n-k)f_1(k)\]

by spatial & temporal homogeneity.

The summand is the nth term of convolution of \(\{f_{r-1}(n)\}\) and \(\{f_1(n)\}\)

Multiply both sides by \(s^n\), sum over \(n\), giving:

\[G_r(s) = G_{r-1}(s)G_1(s)\]

Recursively then, we obtain:

\[G_r(s) = G_1(s)^r\]

Now look at \(G_1(s)\). Now condition on \(X_1\).

\[f_1(n) = P(T_1=n|X_0=0)\]\[= P(T_1=n|X_1=1,X_0=0)P(X_1=1|X_0=0) + P(T_1=n|X_1=-1,X_0=0)P(X_1=-1|X_0=0)\]\[= P(T_1=n|X_1=1,X_0=0)p + P(T_1=n|X_1=-1,X_0=0)(q=1-p)\]\[= P(T_1=n|X_1=1)p + P(T_1=n|X_1=-1)q\]

If \(n>1\), then \(P(T_1=n|X_1=1) = 0\).

Then

\[f_1(n)=P(T_1=n | X_1=-1)q\]\[= f_2(n-1)\]

by temporal and spatial homogeneity.

\[\begin{split}f_1(n) = \cases \left \{f_2(n-1)q \text{ for }n>1 \\ p \text{ for }n=1 \\ 0 \text{ for }n=0 \right\end{split}\]

Then

\[G_1(s) = \sum_{n=0}^\infty s^nf_1(n)\]\[= ps + \sum_{n=0}^\infty s^n f_2(n-1)q\]\[= sp + sq\sum_{n=2}^\infty s^{n-1}f_2(n-1)\]\[= sp + sq\sum_{n=1}^\infty s^n f_2(n)\]\[= sp + sqG_2(s)\]

So,

\[G_1(s) = sp+sqG_2(s) = sp + sqG_1(s)^2\]\[\implies sqG_q(s)^2 - G_1(s) + sp = 0\]\[\implies \frac{q \pm \sqrt{1-4s^2pq}}{2sq}\]

A boundary condition for \(G_1(s)\) is \(G_1(0)=f_1(0)=0\)

The only solution that satisfies the boundary condition is:

\[G_1(s) = \frac{1 - \sqrt{1-4s^2pq}}{2sq}\]

(check using l’Hopital’s rule that this \(\to 0\) as \(s \to 0\))

Now,

\[G_1(1)=\frac{1-\sqrt{1-4pq}}{2q}\]\[=\frac{1-\sqrt{1-4(1-q)q}}{2q}\]\[=\frac{1-\sqrt{1-4q+4q^2}}{2q}\]\[=\frac{1-\sqrt{(1-2q)^2}}{2q}\]\[=\frac{1-|1-2q|}{2q}\]

If \(q<\frac{1}{2}\), then

\[G_1(1)=\frac{1-(1-2q)}{2q}\]\[= \frac{2q}{2q} = 1\]

If \(q<\frac{1}{2}\), then

\[G_1(1)=\frac{1-(2q-1)}{2q}\]\[=\frac{2-2q}{2q}\]\[=\frac{2(1-q)}{2q}\]\[=\frac{2p}{2q}\]\[= \frac{p}{q}\]
