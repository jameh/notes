Theorem

If

\[\underscore X_{n\times 1}\sim N(\underscore μ,k\underscore x_{n\times n})\]

Then

\[h(\underscore X)=\frac{1}{2}\log_2[(2πe)^n(\det K\underscore x)]\]

without loss of generality, \(\underscore μ=0\)

Proof:

Since covariance matrix \(K_{\underscore x}\) is real-valued symmetric, then it is orthogonally diagonalizable:

\(\exists\) \(n\times n\) matrix orthogonal matrix \(A\)

\[(A=A^T)\]

s.t. \(AK_{\underscore x}A^T\) is a diagonal matrix. (with diagonal elements given by the eigenvalues of \(K\underscore x\))

Therefore, for \(\underscore Y=A\underscore X\sim N(\underscore 0, AK_{\underscore x}A^T)\)

where \(AK_{\underscore x}A^T\) is diagonal & Gaussian (\(Y_1,...,Y_n\) are independent Gaussian)

Therefore,

\[\begin{split}h(Y_1,...,Y_n)=h(\underscore Y)\\ =h(Y_1)+h(Y_2)+...+h(Y_n)\text{ since }Y_i\text{ independent}\\ =\sum_{i=1}^n \frac{1}{2}\log_2(2πeVar(Y_i))\text{ }Y_i\text{ is Gaussian}\\ =\frac{n}{2}\log(2πe)+\frac{1}{2}\log[\prod_{i=1}^n Var(Y_i)]\\ =\frac{1}{2}\log(2πe)^n + \frac{1}{2}\log[det K_{\underscore Y}]\\ =\frac{1}{2}\log(2πe)^n + \frac{1}{2}\log[det K_{\underscore X}]\\ =\frac{1}{2}\log_2[(2πe)^n det(K_{\underscore X})]\end{split}\]

Note

\[\begin{split}K_{\underscore Y}=AK_{\underscore X}A^T\\ det(K_{\underscore Y})=det^2 (A)det(K_{\underscore X})=det(K_{\underscore X})\\ |det(A)|=1\end{split}\]

By (1), \(h(\underscore X)=h(\underscore Y)\)

Maximal differential entropy for real-valued random vectors¶

Theorem

Let \(\underscore X_{n\times 1}=(X_1,...,X_n)\) be a real-valued random vector with support \(\mathbb R^n\), mean vector \(\underscore μ\) and covariance matrix \(k_{\underscore X}\) (nxn)

Then

\[\begin{split}h(\underscore X)=h(X_1,...,X_n)\\ \leq \frac{1}{2}\log_2[(2πe)^ndet(K_{\underscore X})]\end{split}\]

with equality iff \(\underscore X\sim N(\underscore μ, K_{\underscore X})\)

Proof:

  1. Scalar case (n=1)

    let \(X\) be real-valued with support \(R\), mean \(μ\), variance \(σ^2\)

    show \(h(X)\leq \frac{1}{2}\log_2(2πeσ^2)\) with equality iff \(X\sim N(μ,σ^2)\)

    Proof:

    Let \(Y\sim N(μ,σ^2)\). Then

    \[\begin{split}0\leq D(X||Y)\\ = \int f_X(t) \log_2\frac{f_X(t)}{\frac{1}{\sqrt{2πσ^2}e^{\frac{a(t-μ)^2}{2σ^2}}}}dt\\ =-h(X)+\int f_X(t)[\log_2(\sqrt{2πσ^2})+\frac{(t-μ)^2}{2σ^2}\log_2 e]dt\\ =-h(X)+ \frac{1}{2}\log(2πσ^2)+\frac{\log_2 e}{2σ^2}\int (t-μ)^2 f_X(t)dt\\ =-h(X)+\frac{1}{2}\log(2πeσ^2)\end{split}\]

    Therefore

    (2)\[h(X)\leq \frac{1}{2}\log(2πeσ^2)\]

    with equality iff \(X\sim N(μ,σ^2)\)

  2. Multivariate case (n>1)

    As in the Pf of the previous theorem, we can use an othogonal matrix \(A_{m\times n}\) (\(A^T=A^{-1}\implies |det(A)|=1\)):

    such that \(AK_{\underscore X}A^T\) is diagonal

    So: for

    \[\underscore Z_{n\times 1}=A\underscore X_{n\times 1}\]

    its covariance matrix \(K_{\underscore Z}=AK_{\underscore X}A^T\) is diagonal.

    Thus, by (1)

    \[\begin{split}h(\underscore X)=h(\underscore Z)-\log|Det A|\\ =h(\underscore Z)\\ =h(Z_1,...,Z_n)\\ \leq \sum_{i=1}^n h(Z_i)\\ \leq \sum_{i=1}^n \frac{1}{2}[2πeVar(Z_i)]\end{split}\]

    Where the last inequality follows from the n=1 case, (2)

    \[= \frac{n}{2}\log_2(2πe)+\frac{1}{2}\log_2[\prod_{i=1}^n Var(Z_i)]\]

    And since \(K_{\underscore Z}\) is diagonal,

    \[\begin{split}= \frac{1}{2}\log_2(2πe)+\frac{1}{2}\log_2(det K_{\underscore Z})\\ = \frac{1}{2}\log[(2πe)^ndet(K_{\underscore X})]\end{split}\]

    since \(|det(A)|=1\)

Theorem (AEP)

If \(X_1,...\) are i.i.d. real-valued random variables with common pdf \(f_X\), then

\[-\frac{1}{n}\log_2 f_{X^n}(X_1,...,X_n)\rightarrow_{n\to\infty}h(X)\]

in probability

Proof: by WLLN

Definition

For \(ε>0\), and any \(n\), define typical set \(A_ε^{(n)}\) with respect to \(f_X\)

\[A_ε^{(n)}=\{x^n\in S_X^n: |-\frac{1}{n}f_X^n(x_1,...,x_n)-h(X)|\leq ε\}\]

where

\[f_{X^n}(x^n)=\prod_{i=1}^n f_X(x_i)\]

\(S_X=\) support of \(X\) = \(\{x\in\mathbb R: f_X(x)>0\}\)

Theorem (Consequence of AEP)

For iid real-valued RVs with pdf \(f_X\), the typical set satisfies:

  1. \[\begin{split}P[A_ε^{(n)}]>1-ε\end{split}\]

    for \(n\) suff. large

  2. \[Vol(A_ε^{(n)})\leq 2^{n(h(X)+ε)}\]
  3. \[\begin{split}Vol(A_ε^{(n)})>(1-ε)2^{n(h(X)-ε)}\end{split}\]

    for \(n\) suff. large.

The Gaussian Channel¶

Definition

Consider a discrete-time channel with continuous input and output alphabets \(\mathcal X\subseteq \mathbb R\), \(\mathcal Y\subseteq \mathbb R\), and a conditional pdf \(f_{Y|X}\). The channel is memoryless if for sending \((x_1,...,x_n)\) over the channel, the output \(y^n=(y_1,...,y_n)\) is governed by

\[f_{Y^n|X^n}(y^n|x^n)=\prod_{i=1}^nf_{Y|X}(y_i|x_i)\]

Additive-noise Gaussian Channel¶

\[Y_i=X_i+Z_i\]

for \(i=1,2,...\)

\(\{Z_i\}\) are iid Gaussian (\(Z_i\sim N(0,N)\)) white Gaussian noise Channel

\[f_{Y|X}=f_Z(y-x)=\frac{1}{\sqrt{2πN}}e^{\frac{-(y-x)^2}{2N}}\]
