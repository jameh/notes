Proof of Shannon’s Channel Coding Theorem for DMCs¶

Consider a DMC \((\mathcal X, \mathcal Y, p_{Y|X}\) with info capacity \(C=max_{p_X}I(X;Y)>0\).

  1. Forward (Achievability) part

    All code rates below \(C\) are achievable - i.e., \(\forall ε>0\) and \(0<R<C\), \(\exists\) a sequence of \((2^{nR},n)\) block codes for the channel such that their \(p_e^{(n)}<ε\) for \(n\) sufficiently large.

  2. Converse part:

    Conversely, if \(R>C\), then it is not an achievable rate (i.e. cannot make \(p_e^{(n)}\) arbitrarily small).

Conclusion:

For a DMC,

\[\text{operational capacity} = C:=max_{p_X}I(X;Y)\]

Remark

A similar theorem can be proced for a large class of “info-stable” channels with memory (i.e. channels for which the input process that maximizes \(\frac{1}{n}I(X^n;Y^n)\) and its corresponding output form a stationary ergodic input-output process)

\[\text{operational capacity}=C=\lim_{n\to\infty}max_{p_{X^n}}\frac{1}{n}I(X^n;Y^n)\]

Proof:¶

  1. Forward Part

    Goal

    Want to prove the existence of a code with block length \(n\), and \(2^{nR}\) codewords, where \(R<C\) such that \(p_e^{(n)}\to 0\) as \(n\to \infty\).

    Technique: Random Coding

    Describe a process of randomly generating a code \(\mathcal C\) with \(2^{nR}\) codewords. Then show that the mean of \(p_e^{(n)}(\mathcal C)\) over all randomly generated codes satisfies:

    \[E_{\mathcal C}[p_e^{(n)}(\mathcal C)]\to 0\text{ as }n\to \infty\]

    This will imply the existence of at least one “good” code, \(\mathcal C^*\) such that \(p_e^{(n)}(\mathcal C^*)\to 0\) as \(n\to\infty\)

    Fix \(R<C\).

    Note

    R can be very close to \(C\), but below it.

    and fix \(n\). (“think of \(n\) as sufficiently large”).

    Let \(p^*_X\) be the input distribution that maximizes the channel’s \(I(X;Y)\), i.e.

    \[I(X;Y)|_{p_X=p_X^*}=C\]

    Generate a \((2^{nR},n)\) code at random according to \(p_X^*\) - i.e. generate each of the \(2^{nR}\) codewords (which a n-tuples over \(\mathcal X\)) by independently drawing n symbols from \(\mathcal X\) according to \(p_X^*\).

    So if \(x^n\in \mathcal X^n\), then the probability that \(x^n\) is chosen as a codeword is:

    \[p^*_X(x^n)=\prod_{i=1}^n p_X^*(x_i)\]

    and let

    \[\mathcal C = \{x^n(1), x^n(2), ..., x^n(2^{nR})\}\]

    be a possible codebook (code), where

    \[x^n(w)=(x_1(w),...,x_n(w))\]

    for \(w=1,2,...,2^{nR}\) is a codeword for message \(w\in \mathcal M=\{1,2,...,2^{nR}\}\)

    Then the probability that \(\mathcal C\) is chosen as our codebook is:

    (1)\[P(\mathcal C)=\prod_{w=1}^{2^{nR}}p_{X^n}^*(x^n(w))=\prod_{w=1}^{2^{nR}}\prod_{i=1}^np_X^*(x_i(w))\]

    How many possible codebooks do we have?

    \[|\mathcal X|^{n2^{nR}}\]

    Consider the following:

    1. A codebook \(\mathcal C\) is generated at random as described via (1) according to \(p_X^*\)

    2. Codebook \(C\) is recealed to both the transmitter and receiver. We also assume that the channel law \(p_{Y|X}\) is known by transmitter/receiver.

    3. A message \(w\) is chosen uniformly from set of messages \(\mathcal M=\{1,2,...,2^{nR}\}\)

      \[\begin{split}P[W=w]=\frac{1}{2^{nR}} && \forall w\in \mathcal M\end{split}\]

    4. Transmit message \(w\) by sending its corresponding codeword \(x^n(w)\) in \(\mathcal C\) over the channel.

    5. At receiver, receive \(y^n\in\mathcal Y^n\) according to:

      \[P[Y^n=y^n|X^n=x^n(w)]=\prod_{i=1}^np_{Y|X}(y_i|x_i(w))\]

      since channel is DMC

    6. Decoding:

      Reciever guesses which message was sent.

      Note

      The optimal decoding rule (\(\forall n\)) is maximum likelihood (ML) decoding

      \[\hat w_{ML}=argmax_{w\in\mathcal M}p(y^n|w)\]

      We use a sub-optimal rule: typical set decoding

      Definition

      Typical Set Decoding

      Decoder rule:

      \[g_{\mathcal C}: \mathcal Y^n\to\{0,1,2,...,2^{nR}\}\]

      Decoder decodes \(g_{\mathcal C}(y^n)=\hat w\) was sent if:

      1. \((x^n(\hat w), y^n)\in A_ε^{(n)}\) jointly typical set \(A_ε^{(n)}\) with \(p_X^*p_{Y|X}\) and
      2. there is no other index \(j\in\mathcal M\) s.t. \((x^n(j),y^n)\in A_ε^{(n)}\), \(\forall j\in\mathcal M,j\neq \hat w\).

      If no such \(\hat w\) exists, declare an error and output 0

    7. Decoding error: \(\hat w\neq w\)
