Rate-Distortion Theory¶

(Lossy Source Coding)

Consider a DMS \(\{X_i\}_{i=1}^\infty\) with finite alphabet \(\mathcal X\sim p_X\)

Definition

For a given \(\mathcal X\), \(\mathcal \hat X\) is the source reproduction alphabet. (usually \(\mathcal X=\mathcal \hat X\), in general \(\mathcal \hat X\subseteq \mathcal X\))

A distortion function, \(d(\cdot,\cdot)\):

\[d:\mathcal X\times \mathcal \hat X\to[0,\infty)\]

\(d(x,\hat x)\) measures the cost of representing source symbol \(x\) via \(\hat x\).

  • bounded

    \[\begin{split}max_{x,\hat x}d(x,\hat x)<\infty\end{split}\]

Common Distortion Measures¶

Hamming Distortion measure¶

\[\begin{split}d(x,\hat x)=\begin{cases} 1\text{ if }x\neq \hat x\\ 0\text{ if }x=\hat x \end{cases}\end{split}\]
\[E[d(X,\hat X)]=P[X\neq \hat X]\]

Squared Error¶

\[d(x,\hat x)=(x-\hat x)^2\]

Additive distortion measure¶

\[d(x^n,\hat x^n)=\frac{1}{n}\sum_{i=1}^n d(x_i,\hat x_i)\]

Definition

A \((2^{nR},n)\) rate-distortion (lossy source) code encoder:

\[f_n: \mathcal X^n\to\{1,2,3,...,2^{nR}}\]

decoder:

\[g_n: \{1,2,3,...,2^{nR}}\to\hat \mathcal X\]

Distortion associated with code:

\[\frac{1}{n}E[d(X^n,g_n(f_n(X^n)))]=\frac{1}{n}\sum_{\mathcal X^n}p_{X^n}(x^n)d(x^n,g_n(f_n(x^n)))\]

Codebook:

\[\mathcal C=\{g_n(1),...g_n(2^{nR})\}=\{\hat x^n(1),...,\hat x^n(2^{nR})}\]

Definition

Operational Rate-Distortion function:

Given \(D\geq 0\), the operational rate-distortion function \(R(D)\) for a source is the infimum of all rates \(R\) for which \(\exists\) a sequence of \((2^{nR},n)\) rate-distortion codes \((f_n,g_n)\) with

\[\lim_{n\to\infty}E[d(X^n,g_n(f_n(X^n)))]\leq D\]

Definition

Information Rate-distoriton function

\[R^{(I)}(D):=min_{p_{\hat X|X}: E[d(X,\hat X)]\leq D} I(X;\hat X)\]

Theorem (Shannon’s Rate-Distortion Theorem for DMS)

\[R(D)=R^{(I)}(D):=min_{p_{\hat X|X}: E[d(X,\hat X)]\leq D} I(X;\hat X)\]

where \(R(D)\) is the operational rate-distortion function

See Text for Proof

Example (IID Bernoulli source)

\(\{X_i\}\) is a binary-valued iid source with \(P[X=0]=p\). Consider Hamming distortion measure

\[\begin{split}R(D)=\begin{cases} h_b(p)-h_b(D), & 0\leq D\leq min(p,1-p)\\ 0 & \text{otherwise} \end{cases}\end{split}\]
\[h_b(x)=-x\log x-(1-x)\log(1-x)\]

Note

When \(D=0\), \(R(D)=h_b(p)=H(X)\) - reduces to source coding Theorem.

Example (Gaussian Source)

IID Gaussian source \(\{X_i\}\)

\[X_i\sim N(0,σ^2)\]

and squared dist. measure \(d(x,\hat x)=(x-\hat x)^2\)

\[\begin{split}R(d)=\begin{cases} \frac{1}{2}\log_2 \frac{σ^2}{D} & \text{ if }0<D\leq σ^2\\ 0 & \text{ if }D>σ^2 \end{cases}\end{split}\]

Lossy Joint Source-Channel Coding Theorem¶

  1. If a \(rR(D)<C\), then communication of source over the channel via a lossy-JSCC at rate \(r\) is possible with asymptotic exected distortion \(\leq D\)
  2. If \(rR(D)>C\), not possible.
