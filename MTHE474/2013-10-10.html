Kraft Inequality & Lossless Variable-Length Source-Coding¶

Definition

A set of positive integers \(\{l_1,l_2,...l_n\}\) is said to satisfy the Kraft (or Kraft-McMillan) inequality if:

\[\sum_{i=1}^M D^{-l_i}\leq 1\]

Where \(D\) is a given integer, \(D\geq 2\)

Theorem

Kraft Inequality for Uniquely Decodable Codes:

A UD Variable length code (VLC) \(\mathcal C\) with D-ary code alphabet and \(M\) codewords having lengths \(l_1,l_2,...,l_M\) (\(M=|\mathcal X|^n\)) (where \(f:\mathcal X^n\to\{0,1,...D-1\}\))

must satisfy the Kraft inequality:

\[\sum_{i=1}^M D^{-l_i}\leq 1\]

Suppose that we use the code \(\mathcal C\) to encode \(\mathcal N\) n-tuples, that is:

\[x_i^n\in\mathcal X^n,i=1,...,N\]

arriving in sequence.

This yields a concatenated codeword sequence:

\[f(x_1^n)f(x_2^n)\cdot\cdot\cdot f(x_N^n)=c_1c_2\cdot\cdot\cdot c_N\]

Let the lengths of these codewords respectively be given by:

\[l(c_1),l(c_2),...,l(c_N)\]

Consider the following expression:

\[\sum_{c_1\in\mathcal C}\sum_{c_2\in\mathcal C}...\sum_{c_N\in\mathcal C}D^{(l(c_1)+l(c_n)+...+l(c_N))} (*)\]

It is clear the the above expression can be written as:

\[\left[\sum_{c\in \mathcal C}D^{-l(c)}\right]^N\]\[=\left[\sum_{m=1}^M D^{-lm}\right]^N (**)\]

where \(|\mathcal C|=M\)

Now, \(l(c_1)+l(c_2)+...+l(c_n)\) is the total length of the codeword sequence \(c_1,c_2,...,c_N\):

It can take values anywhere between \(NL_1\) and \(NL_2\) where

\[L_1:=min_{c\in\mathcal C}\{l_1,...,l_M\}\]\[L_2:=max_{c\in\mathcal C}\{l_1,...,l_M\}\]

Define \(A_j\) as the number of N-codeword sequences of total length \(j\). Then, by collecting the terms in (*), we have the following for the right-hand side term in (**):

\[\left[\sum_{m=1}^M D^{-lm}\right]^N=\sum_{j=NL_1}^{NL_2} A_j D^{-j}\]

Since the code \(\mathcal C\) is UD (by assumption), we have that sequences of codewords of total length \(j\) must be distinct (unambiguously decodable). But the maximum possible number of distinct such sequences of length \(j\) id \(D^j\).

Therefore, we must have that \(A_j\leq D^j\)

Proceeding, we have:

\[\sum_{j=NL_1}^{NL_2} A_j D^{-j}\leq \sum_{j=NL_1}^{NL_2} D^j D^{-j}=N(L_2-L_1)+1\]\[\leq NL_2\]

(since \(L_1\geq 1\))

Therefore,

\[\sum_{n=1}^M D^{-ln}\leq (NL_2)^{\frac{1}{N}}=e^{\frac{1}{N}ln(NL_2)}\]

Note the above inequality hols \(\forall N\geq 1\); it also holds in the limit on \(N\to\infty\). But as \(N\to\infty\),:math:e^{frac{1}{N}ln(NL_2)}to 1

Therefore,

\[\sum_{m=1}^M D^{-lm}\leq 1\]

Theorem

  • Every D-ary prefix variable length code with \(M\) codeword lengths \(l_1,_2,...l_M\) satisfies the Kraft inequality.
  • Conversely, given a set of lengths \(l_1,_2,...l_M\) that satisfy the Kraft inequality, there exists a D-ary prefix code with these lengths as its codeword lengths

see online notes (Theorem 3.20)

Big Picture¶

Conclusion of previous two theorems:¶

A UD VLC can always be replaced with a prefix code of identical codeword lengths (and thus identical rates).

Theorem

Consider a DMS \(\{X_i\}_{i=1}^\infty\) with finite alphabet \(\mathcal X\) and entropy in base \(D\), \(H_D(X)=-\sum_{x\in\mathcal X}p_X(x)log_D p_X(x)\)

For any \(\epsilon>0\), there exists a prefix (hence UD) code for the source:

\[f:\mathcal X^n\to\{0,1,...,D-1\}^*\]

satisfying

\[\begin{split}\bar R<H_D(X)+\epsilon\end{split}\]

for \(n\) sufficiently large.

Conversely, every UD VLC for the source

\[f:\mathcal X^n\to\{0,1,...,D-1\}^*\]

has average code rate:

\[\bar R\geq H_D(X)\]
