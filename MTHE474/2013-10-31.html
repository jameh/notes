The Channel Coding Theory¶

Definition

Block code for discrete channels

Consider a discrete channel \((\mathcal X,\mathcal Y,P_{Y^n|X^n})\). Given a blocklength \(n\) and a set of messages \(\mathcal M=\{1,2,3,...,M\}\) with \(M=M(n)\), then an \((M,n)\) block code for the channel consists of:

  • Encoding function \(f: \mathcal M\to \mathcal X^n\)

    The encoding function yields codewords

    \[f(1)=x^n(1),f(2)=x^n(2),...,f(M)=x^n(M)\]

    where each codeword \(x^n(i)=(x_1(i),...,x_n(i)), i\in \mathcal M\).

    The set of these \(M\) codewords of blocklength \(n\) is called the codebook:

    \[\mathcal C=\{x^n(1),...,x^n(M)\}\subseteq \mathcal X^n\]

    To transmit message \(W\) over the channel, the encoder sends codeword \(f(W)=X^n(W)\) over the channel.

  • Decoding function \(g: \mathcal Y^n\to \mathcal M\)

    Receive \(Y^n\) at the channel output and estimate the transmitted messages via \(g(Y^n)=\hat W\)

    The code rate

    \[\begin{split}R&=\frac{1}{n}\log_2|\mathcal M| &=\frac{1}{n}\log_2M\frac{\text{message bits}}{\text{channel symbol (use)}}\end{split}\]

Definition

Probability of Error

If message \(i\in\mathcal M\) is sent over the channel (via its codeword \(x^n(i)\)), an error occurs at the decoder if \(g(y^n)\neq i\).

The conditional probability of decoding error given message \(i\) is sent is:

\[\begin{split}λ_i:=&P[g(Y^n)\neq i|X^n=x^n(i)] =&\sum_{y^n\in\mathcal Y^n: g(y^n)\neq i}P_{Y^n|X^n}(y^n|x^n(i))\end{split}\]

where \(P_{Y^n|X^n}(\cdot)\) is the channel probability law

Error Criteria

  1. Maximal probability of error: \(λ^{(n)}\) for an \((M,n)\) code:

    \[λ^{(n)}:=max_{i\in\mathcal M=\{1,2,...,M\}}λ_i\]

  2. Average probability of error: \(P_e^{(n)}\) for an \((M,n)\) code:

    \[P_e^{(n)}=\frac{1}{M}\sum_{i=1}^Mλ_i\]

Note

Obviously, \(P_e^{(n)}\leqλ^{(n)}\).

Note

If message \(W\) is uniformly distributed over \(\mathcal M=\{1,2,...,M\}\), then

\[P_e^{(n)}=P[g(Y^n)\neq W]\]

Remark

One would expect that \(P_e^{(n)}\) behaves differently than \(λ^{(n)}\). But we will see that (for the class of channels considered) a small \(P_e^{(n)}\) implies a small \(λ^{(n)}\) at essentially the code rate. We will say that a channel code is “good” if either \(P_e^{(n)}\) or \(λ^{(n)}\) can be made arbitrarily small for large \(n\).

Definition

Achievable Rate

\(R>0\) is called an achievable code rate for a channel if there exists a sequence of \((\lceil 2^{nR}\rceil,n)\) block codes for the channel such that the code’s maximized error probability

\[λ^{(n)}\to 0\text{ as }n\to\infty\]

(i.e. for arbitrary \(ε>0\), \(\exists\) \(n_0=n_0(ε,R)\) s.t. \(\forall n\geq n_0\), there exists a code of blocklength \(n\) and rate \(R\) with \(λ^{(n)}\leq ε\)).

Note

For ease of notation, we write \((2^{nR},n)\) code to mean \((\lceil 2^{nR}\rceil,n)\)

Definition

Operational capacity

The operational capacity of a channel is the supremum of all achievable code rates.

(i.e. it is the largest rate at which one can transmit info over the channel via block codes and recover it at the decoder reliably (i.e. with vanishing error probability for \(n\) large).)

Shannon’s Discovery:¶

It is possible to have arbitrarily small error probability without decreasing rate (at a fixed rate) for \(n\) large.

\(\implies\) such good codes exist

For a DMC,

\[\text{operational capacity}=C:=max_{p_X}I(X;Y)\]

Definition

Joint Typicality

The set \(A_ε^{(n)}\) of jointly typical sequences \(\{(x^n,y^n\}\) with respect to distribution \(p_{XY}(x,y)\) is the set of n-sequences whose empirical entropies are ε-close to the time entropies:

\[A_ε^{(n)}:=\{(x^n,y^n)\in\mathcal X^n\times\mathcal Y^n: |-\frac{1}{n}\log_2p_{X^n}(x^n)-H(X)|\leq ε, |-\frac{1}{n}\log_2p_{Y^n}(y^n)-H(Y)|\leq ε, |-\frac{1}{n}\log_2p_{X^n,Y^n}(x^n,y^n)-H(X,Y)|\leq ε\}\]

where

\[\begin{split}p_{X^nY^n}(x^n,y^n)&=\prod_{i=1}^n p_{XY}(x_i,y_i)\\ p_{X^n}(x^n)&=\prod_{i=1}^np_X(x_i)\\ p_{Y^n}(y^n)&=\prod_{i=1}^np_Y(y_i)\end{split}\]
