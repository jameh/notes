General channel with Memory¶

  • Finite input alphabet \(\mathcal X\)
  • Finite output alphabet \(\mathcal Y\)
  • \(p_{Y^n|X^n}(y^n|x^n)\), \(n=1,2,...\), \(x^n\in \mathcal X^n\), \(y^n\in \mathcal Y^n\)

Definition (DMC)

A discrete memoryless channel (DMC) is a discrete channel for which the conditional (transition) distribution of output given input satisfies:

(1)\[\begin{split}p_{Y^n|X^n}(y^n|x^n)=\prod_{i=1}^n p_{Y|X}(y|x) && i=1,2...\\ (x^n,y^n)\in \mathcal X \times \mathcal Y\end{split}\]

where \(p_{Y|X}(y|x)\), \((x^n,y^n)\in \mathcal X \times \mathcal Y\) is a one-dimensional transition distribution

Remark

The following equivalences can be shown:

(Observation 4.3 online notes)

The above condition for a DMC, (1), is equivalent to (a) and (b) together, while (b) \(\iff\) (c):

  1. \(P_{Y_n|X^n,Y^{n-1}}(y_n|y^{n-1},x^{n})=P_{Y|X}(y_n|x_n)\) (memoryless property)
  2. \(P_{X_n|X^{n-1},Y^{n-1}}(x_n|x^{n-1},y^{n-1})=P_{X_n|X^{n-1}}(x_n|x^{n-1})\) (non-anticipatory property)
  3. \(P_{Y^{n-1}|X^n}(y^{n-1}|x^n)=P_{Y^{n-1}|X^{n-1}}(y^{n-1}|x^{n-1})\)

Examples of DMC

  1. Binary Symmetric Channel (BSC)

    \[\mathcal X=\mathcal Y=\{0,1\}\]
    \[\begin{split}Q_{|\mathcal X|\times |\mathcal Y|}&=[p_{Y|X}(y|x)]\\ &=\left[\matrix{p_{Y|X}(0|0) && p_{Y|X}(1|0)\\ p_{Y|X}(0|1) && p_{Y|X}(1|1)}\right]\\ &=\left[\matrix{1-ε && ε\\ ε && 1-ε}\right]\end{split}\]

    where \(ε\) is the crossover probability

    Remark

    BSC can be described as a modulo-2 additive noise channel via:

    \[\begin{split}Y_n=X_n\oplus Z_n &&, m=1,2,...\end{split}\]

    • \(\oplus\) modulo-2 addition

    • \(\{Z_n\}^\infty_{n=1}\) iid noise process, binary-valued

      with distribution \(P[Z_n=1]=ε\), \(\forall n=1,2,...\)

    • Input and noise processes are independent of each other.

    Note

    When \(ε=0\), the channel is noiseless.

  2. Binary Erasure Channel (BEC)

    \[\begin{split}\mathcal X=\{0,1\} && \mathcal Y=\{0,E,1\}\end{split}\]
    \[\begin{split}Q_{|\mathcal X|\times |\mathcal Y|}&=[p_{Y|X}(y|x)]\\ &=\left[\matrix{p_{Y|X}(0|0) && p_{Y|X}(1|0)\\ p_{Y|X}(0|1) && p_{Y|X}(1|1)}\right]\\ &=\left[\matrix{1-α && α\\ α && 1-α}\right]\end{split}\]

    Where \(α\) is the erasure probability

  3. A more general channel BSEC

    \[\begin{split}\mathcal X=\{0,1\} && \mathcal Y=\{0,E,1\}\end{split}\]
    \[\begin{split}Q_{|\mathcal X|\times |\mathcal Y|}&=[p_{Y|X}(y|x)]\\ &=\left[\matrix{p_{Y|X}(0|0) && p_{Y|X}(1|0)\\ p_{Y|X}(0|1) && p_{Y|X}(1|1)}\right]\\ &=\left[\matrix{1-α-β && α && β\\ β && 1-α-β && α}\right]\end{split}\]

Fundamental Problem¶

Need to transmit over the channel information at a high rate such that decodability at the receiver is possible with arbitrary low probability of error

Definition

Given a DMC \((\mathcal X,\mathcal Y,p_{Y|X})\), its “information” capacity denoted by \(C\) is defined as:

\[C:=max_{p_X}I(X;Y)\]

where \(X\) is the input of the channel

and \(Y\) is the output of the channel

maximization is taken over all input distributions \(p_X\)

Remark

    • \(C\geq 0\), since \(I(X;Y)\geq0\)
    • \(C\leq min(log_2|\mathcal X|, log_2|\mathcal Y|)\) bounded
    • \(I(X;Y)\) is concave (and continuous in the variational distance [1]) in \(p_X\) (for a fixed \(p_{Y|X}\))
    • \(C\) is well-defined: there exists at least one input distribution \(p_X^*\) that maximizes \(I(X;Y)\)

  2. For a channel, its “operational” capacity will be defined as the highest rate in bits/channel symbol at which information can be transmitted over the channel via block code and recovered at the receiver with arbitrarily low error probability.

    We will see via Shannon’s channel coding Theorem, that for a DMC,

    operational capacity = “info capacity”= \(C=max_{p_X}I(X;Y)\)

[1]\(d_{var}(p_X,p_{X'})=\sum_{\mathcal X}|p_{X'}(x)-p_X(x)|\)
