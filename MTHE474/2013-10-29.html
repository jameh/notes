Cool capacity stuff¶

For a weakly-symmetric DMC \((\mathcal X,\mathcal Y,Q)\)

\[\begin{split}C:=&max_{p_X}I(X;Y)\\ =&log_2 |\mathcal Y| - H(q_1,q_2,...,q_{|\mathcal Y|})\\ =&log_2 |\mathcal Y| - \sum_{i=1}^{|\mathcal Y|}q_ilog_2q_i\end{split}\]

where \((q_1,...,q_{|\mathcal Y|})\) is any row in \(Q\).

Proof¶

\[\begin{split}I(X;Y)=H(Y)-H(Y|X)\\ H(Y|X)=\sum_{x\in\mathcal X}p_X(x)H(Y|X=x)\\\end{split}\]

where

\[H(Y|X=x)=-\sum_{y\in\mathcal Y}p_{Y|X}(y|x)log_2 p_{Y|X}(y|x)\]

Since every row of \(Q\) is a permutation of every other row, \(H(Y|X=x)\) is independent of \(x\):

\[H(Y|X=x)=H(q_1,...,q_{|\mathcal Y|})\]

Therefore,

\[H(Y|X)=H(q_1,...,q_{|\mathcal Y|})\]

Hence,

\[\begin{split}I(X;Y)&=H(Y)-H(q_1,...,q_{|\mathcal Y|})\\ &\leq log_2 |\mathcal Y|-H(q_1,...,q_{|\mathcal Y|})\end{split}\]

with equality iff \(Y\) is uniform.

But choosing \(p_X(x)=\frac{1}{|\mathcal X|}\) for all \(x\in \mathcal X\) (uniform) yields a uniformly distributed output Y since:

For \(y\in \mathcal Y\),

\[\begin{split}p_Y(y)&=\sum_{x\in\mathcal X}p_X(x)p_{Y|X}(y|x)\\ &=\frac{1}{|\mathcal X|}\sum_{x\in\mathcal X}p_{Y|X}(y|x)\\ &=c\text{ (constant)}\\ &=\text{sum of entries in column in }Q\text{ for value y}\end{split}\]

since:

\[\sum_{y\in\mathcal Y}p_Y(y)=1\iff c\frac{|\mathcal Y|}{|\mathcal X|}=1\iff c=\frac{|\mathcal Y|}{|\mathcal X|}\]

therefore,

\[\begin{split}p_Y(y)&=\frac{|\mathcal X|}{|\mathcal Y|}\cdot\frac{1}{|\mathcal X|}\\ &=\frac{1}{|\mathcal Y|} && \forall y\in \mathcal Y\end{split}\]

therefore, \(Y\) is uniform.

Example

  1. BSC(\(ε\)): Applying (*),

    \[log_2|\mathcal Y|=H(1-ε,ε) =1-h_b(ε)\]

  2. q-ary symmetric channel:

    \[C=log_2 q - H(1-ε, \frac{ε}{q-1},...,\frac{ε}{q-1})\]

Definition

Quasi-symmetric DMC

A DMC \((\mathcal X,\mathcal Y,Q_{|\mathcal X|\times|\mathcal Y|}=[p_{Y|X}])\) is said to be Quasi-symmetric if \(Q\) can be partitioned along its columns into \(m\) weakly-symmetric sub-matrices \(Q_1,Q_2,...,Q_m\) (for some integer \(m\geq 1\)), where each sub-matrix \(Q_i\) has size \(|\mathcal X| \times |\mathcal Y_i|\), \(i=1,...,m\) with

\[\begin{split}\mathcal Y_1\cup \mathcal Y_2\cup ...\cup \mathcal Y_m=\mathcal Y\\ \mathcal Y_i\cap Y_j=\{\} && \forall i\neq j\end{split}\]

Example

BEC(\(α\)):

\[\begin{split}Q=\left[\matrix{1-α && α && 0 \\ 0 && α && 1-α }\right]\end{split}\]
\[\begin{split}Q_1&=\left[\matrix{1-α && 0 \\ 0 && 1-α}\right]\\ Q_2&=\left[\matrix{α\\α}\right]\end{split}\]

Note

symmetry \(\implies\) weak symmetry \(\implies\) quasi-symmetry

Theorem

Capacity of quasi-symmetric DMC:

The capacity of a quasi-symmetric DMC \((\mathcal X, \mathcal Y, Q)\) is given by:

\[C=\sum_{i=1}^m a_iC_i\]

where

\[\begin{split}a_i=\sum_{y\in\mathcal Y_i}p_{Y|X}(y|x)=\text{sum of any row in sub-matrix }Q_i\\ C_i=log_2|\mathcal Y_i|-H(\text{any row in matrix }\frac{1}{a_i}Q_i)\end{split}\]

\(i=1,...,m\)

see online notes Lemma 4.17

\[\begin{split}C=a_1C_1+a_2C_2\\ a_1=1-α\\ a_2=α\\ C_1&=log_2 2- H(\frac{1-α}{1-α}\frac{0}{1-α})\\ &=1-H(1,0)=1\\ C_2=log_2|1| - H(\frac{α}{α})=0\\ C=(1-α)(1)+(α)(0)\\ \implies C=1-α\end{split}\]

Example

BEC

\[\begin{split}Q=\left[\matrix{1-α-ε && α && ε\\ ε && α && 1-α-ε}\right]\end{split}\]

is quasi-symmetric:

\[\begin{split}Q_1=\left[\matrix{1-α-ε && ε\\ ε && 1-α-ε}\right]\\ Q_2=\left[\matrix{α\\α}\right]\end{split}\]

Apply previous theorem to get:

\[C=(1-α)[1-h_b((1-ε-α)/(1-α))]\]

special case:

\[\begin{split}α&=0\implies BSC(ε)\\ ε&=0\implies BEC(α)\end{split}\]

Capacity of sum channel¶

\[\begin{split}Q=\left[\matrix{Q_{1_{|\mathcal X_1|\times |\mathcal Y_1|}} && 0\\ 0 && Q_{2_{|\mathcal X_2|\times |\mathcal Y_2|}} } \right]\end{split}\]
\[\begin{split}\mathcal X=\mathcal X_1\cup \mathcal X_2\\ \mathcal Y=\mathcal Y_1\cup \mathcal Y_2\end{split}\]

with

\[\begin{split}\mathcal X_1\cap \mathcal X_2=\{\}\\ \mathcal Y_1\cap \mathcal Y_2=\{\}\end{split}\]
\[C=log_2(2^{C_1}+1^{C_2})\]

where \(C_1\) is the capacity of \(Q_1\) and \(C_2\) is the capacity of \(Q_2\).
