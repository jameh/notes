Proof of Lossless Variable-Length Source-Coding Theorem for DMS¶

Proof¶

  1. Forward Part: For each source n-tuple \(x^n\in\mathcal X^n\), choose the length \(l_{X^n}\) of its associated codeword according to:
\[l_{X^n}=\lceil -\log_D p_{X^n}(x^n)\rceil\]

where \(\lceil x\rceil\) is the smallest integer \(\geq x\)

Now:

\[l_{X^n}=\lceil -\log_Dp_{X^n}(x^n)\rceil\implies l_{X^n}\geq-\log_D p_{X^n}(x^n)\]\[\implies D^{-lx^n}\leq p_{X^n}(x^n)\]\[\implies \sum_{x^n\in\mathcal X^n}D^{-lx^n}\leq\sum_{x^n\in\mathcal X^n}p_{X^n}(x^n)=1\]

Therefore,

\[\sum_{x^n\in\mathcal X^n}D^{l_{x^n}}\leq 1\]

and Kraft’s inequality is satisfied for these lengths.

Therefore, there exists a D-ary prefix code for the source with the above codeword lengths (by previous theorem)

Furthermore, for this prefix code,

\[\begin{split}l_{X^n}=\lceil -\log_D p_{X^n}(x^n)\rceil\implies l_{X^n}<-\log_D p_{X^n}(x^n)+1\end{split}\]\[\begin{split}\implies \sum_{x^n\in\mathcal X^n}p_{X^n}(x^n)l_{X^n}<-\sum_{x^n\in\mathcal X^n}p_{X^n}(x^n)\log_D P_{X^n}(x^n)+\sum_{X^n}p_{X^n}(x^n)\end{split}\]\[=H_D(X^n)+1\]

Therefore,

\[\begin{split}\bar l<H_D(X^n)+1\end{split}\]

Therefore,

\[\begin{split}\bar R=\frac{\bar l}{n}<\frac{H_D(X^n)}{n}+\frac{1}{n}\end{split}\]

Since the source is a DMS, we have that

\[H_D*X^n)=nH_D(X)\]

Therefore, for a DMS,

\[\begin{split}\bar R<H_D(x)+\frac{1}{n}\end{split}\]

choosing \(n\) large enough (s.t. \(\frac{1}{n}<\epsilon\)) completes the proof.

  1. Converse Part:

Consider a UD code \(f:\mathcal X^n\to\{0,...,D-1\}^*\)

and let \(l_{X^n}\) be the length of codeword \(f(x^n)\) associated to \(x^n\). Then:

\[H_D(X)-\bar R=\frac{1}{n}\left[\frac{nH(X)}{\log_2 D}-\bar l\right]\]

where \(H_D(X)\) is in base \(D\) and \(H(X)\) is in base 2.

\[=\frac{1}{n\log_2 D}\left[nH(X)-\bar l \log_2 D\right]\]\[=\frac{1}{n\log_2 D}\left[H(X^n)-\bar l \log_2 D\right]\]

Since source is DMS

\[=\frac{1}{n\log_2 D}\left[-\sum_{X^n}p_{X^n}p_{X^n}(x^n)\log_2 p_{X^n}(x^n)-\sum_{X^n}p_{X^n}(x^n)l_{x^n} \log_2 D\right]\]\[=\frac{1}{n\log_2 D}\left[-\sum_{X^n}p_{X^n}p_{X^n}(x^n)\log_2 p_{X^n}(x^n)+\sum_{X^n}p_{X^n}(x^n) \log_2 D^{-l_{x^n}}\right]\]\[=\frac{1}{n\log_2 D}\left[\sum_{X^n}p_{X^n}(x^n)\log_2\frac{D^{-l_{x^n}}}{p_{X^n}(x^n)}\right]\]\[\leq \frac{1}{\ln 2}\frac{1}{n\log_2 D}\left[\sum_{X^n}p_{X^n}(x^n)(\frac{D^{-l_{x^n}}}{p_{X^n}(x^n)}-1)\right]\]\[=\frac{1}{\ln 2}\frac{1}{n\log_2 D}\left[\sum_{X^n}D^{-l_{x^n}}-1\right]\]

But since the code is UD, it must satisfy Kraft’s inequality:

\[\sum_{X^n}D^{-l_{x^n}}\leq 1\]

Therefore, for any \(n\), every UD code \(f:\mathcal X^n\to\{0,1,...D-1\}^*\) satisfies:

\[H_D(X)-\bar R\leq 0\]\[\iff \bar R\geq H_D(X)\]

note

Note that \(\bar R=H_D(X)\) iff \(p_{X^n}(x^n)=D^{-l_{X^n}}\) for all \(x^n\in\mathcal X^n\) (D-adic distribution)

Conclusion:

For DMS, the source entropy \(H_D(X)\) measured in D-ary code symbols/source symbol, is the smallest variable length lossless compression rate (for \(n\) sufficiently large).

Corollary

For any \(n\), there exists a prefix (hence UD) VLC code

\[f: \mathcal X^n\to\{0,1,...D-1\}^*\]

for a DMS \(\{X_i\}_{i=1}^\infty\) s.t.

\[\begin{split}H_D(X)\leq \bar R<H_D(X)+\frac{1}{n} (*)\end{split}\]

For a general source, the above statement holds with (*) replaced by:

\[\begin{split}\frac{H_D(X^n)}{n}\leq \bar R<\frac{H_D(X^n)}{n}+\frac{1}{n}\end{split}\]

Theorem

Lossless Variable-Length Source Coding Theorem for Sources with Memory: stationary sources

Consider a stationary source \(\{X_i\}_{i=1}^\infty\) with finite alphabet \(\mathcal X\), and entrpy rate \(H_D(\mathcal X)\). Then:

  1. For any \(\epsilon>0\), there exists prefix (hence UD) code \(f:\mathcal X^n\to\{0,1,....,D-1\}^*\) satisfying:
\[\begin{split}\bar R<H_D(\mathcal X)+\epsilon\text{ for n suff. large}\end{split}\]
  1. Conversely, every UD code \(f:\mathcal X^n\to\{0,1,....,D-1\}^*\) for the source satisfies:
\[\bar R\geq H_D(\mathcal X)\]

Conclusion:

The entropy rate \(H_D(\mathcal X)\) for a stationary source is the smallest CL compression rate for which there exists D-ary prefix codes (for \(n\) suff. large).

Construction of Optimal VLC: Huffman Codes¶

Goal:¶

Given a DMS with alphabet \(\mathcal X=\{a_1,a_2,...a_M\}\) and dist. vector (pmf) \((p_1,p_2,...p_M)\) where \(p_i=p_X(a_i)\), \(i=1,...M\) where \(\sum_{i=1}^M p_i=1\), \(M=|\mathcal X|\)

Construct a binary UD VLC

\[f:\mathcal X^n\to\{0,1\}^*\]

such that the codes average codeword length \(\bar l\) (or equivalently the average code rate \(\bar R=\frac{\bar l}{n}\)) is minimized.

Such a code is called optimal.

Notes

  • we will first construct optimal codes for \(n=1\) (\(f:\mathcal X\to\{0,1\}\)). Finding codes for \(n>1\) follows directly by considering \(\mathcal X^n\) as a “super-alphabet”.

  • for \(n=1\), we know already that \(\exists\) a binary prefix code for the source with

    \[\begin{split}H(X)\leq \bar l<H(X)+1\end{split}\]

    We want to find an *optimal such code (i.e. with \(\bar l\) minimal)

    A code with \(\bar R=H(X)\) is called absolutely optimal
