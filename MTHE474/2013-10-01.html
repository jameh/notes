Entropy Rate for Sources with Memory¶

A general discrete source \(\{X_i\}_{i=1}^\infty\) with memory, and of alphabet \(\mathcal X\), is described by the sequence of \(n\) - fold joint pmfs:

\[p_{X^m}(x_1,x_2,...,x_n)=P[X_1=x_1,X_2=x_2,...,X_n=x_n]\]\[n = 1,2,...\]\[x^n = (x_1,...,x_n) \in \mathcal X^n\]

Note

\[p_{X^n}(x^n)=p_{X_1}(x_1)\prod_{i=2}^n p_{X_i|X^{i-1}}(x_i|x_{i-1},...,x_1)\]

Definition

A stochastic process (or source) \(\{X_i\}_{i=1}^\infty\) is called stationary (or strictly stationary) if:

(1)\[P[X_1=x_1,...,X_n=x_n]=P[X_{1+\tau}=x_1,...,X_{n+\tau}=x_n]\]

The idea: joint \(n\) - fold dist. of the source is invariant under time shifts:

i.e. \(X_1,X_2,...X_n\) has the same distribution as \(X_{1+\tau},X_{2+\tau},...,X_{n+\tau}\).

In practice, many sources are modeled via stationary processes.

Note

  1. source stationarity \(\implies\) source identically distributed (same marginal)
\[P[X_1=a] = P[X_{1+\tau}=a]\]\[\forall \tau, \forall a \in \mathcal X\]

This follows by using (1) with \(n=1\).

  1. A DMS (which consists of iid RVs) is stationary.
\[P[X_1=x_1,X_2=x_2,...,X_n=x_n]\]\[=\prod_{i=1}^\infty p_{X_i}(x_i)\]

by independence

\[= \prod_{i=1}^n p_{X_{i+\tau}}(x_i)\]

by identical distribution

\[= P[X_{1+\tau}=x_1,...,X_{n+\tau}=x_n]\]

by independence.

Overview of Markov chains (Markov sources)¶

Definition

A discrete process \(\{X_i\}_{i=1}^\infty\) with finite alphabet \(\mathcal X\) is called a Markov source or a Markov chain (MC) if for \(n=1,2,...\)

\[P[X_n=x_n|X_{n-1}=x_{n-1},...,X_1=x_1]=P[X_n=x_n|X_{n-1}=x_{n-1}]\]

for all \(x^n=(x_1,...,x_n) \in \mathcal X^n\)

In this case:

\[p_{X^n}(x^n)=p_{X_1}(x_1)\prod_{i=2}^n p_{X_i|X_{i-1}}(x_i|x_{i-1})\]

Also, \(\{X_i\}\) is called a Markov source of order \(M\) (or memory \(M\)), where \(M>0\) an integer - if:

\[P[X_n=x_n|X_{n-1}=x_{n-1},...X_1=x_1]=P[X_n=x_n|X_{n-1}=x_{n-1},...,X_{n-m}=x_{n-m}]\]

for all \(n>M\) and \(x^n \in \mathcal X^n\)

Definition

A MC \(\{X_i\}\) is homogeneous or time-invariant if \(p_{X_n|X_{n-1}}\) does not depend on \(n\) for all \(n\) - i.e.

(2)\[p_{X_n|X_{n-1}}(a|b):=P[X_n=a|X_{n-1}=b]=P[X_2=a|X_1=b]=p_{X_2|X_1}(a|b)\]

Definition

For a MC \(\{X_i\}\), \(X_i\) is called the “state” of the MC at time \(i\).

Note

Using (1) and (2), we remark that a homogeneous MC is fully described by its initial state distribution

\[p_{X_1}(x)=P[X_1=x], x \in \mathcal X\]

and its conditional distribution \(p_{X_2|X_1}\) which can be expressed via a so-called transition probability matrix \(|\mathcal X|\) by \(|\mathcal X|\):

\[Q = [p_{ij}]\]

where \(p_{ij} := P[X_2=j|X_1=i]\)

Definition

A MC \(\{X_i\}\) is irreducible if one can go from any state value in \(\mathcal X\) to any other state value in \(\mathcal X\) in a finite number of steps (transitions) with positive probability. i.e.

\[\forall b,c \in \mathcal X,i=1,2,...\]\[\begin{split}\exists \text{ integer } t>0 \text{ s.t.}\end{split}\]\[\begin{split}P[X_{i+t}=b|X_i=c]>0\end{split}\]

Example

\[\mathcal X = \{0,1,2\}\]

irreducible

not irreducible (reducible)

Definition

For a MC \(\{X_i\}\) with finite alphabet

\[\mathcal X = \{1,2,3,...,m\}\]

and with transition probability matrix

\[Q=[p_{ij}]\]

then the probability vector

\[\Pi:=(\Pi_1,\Pi_2,...,\Pi_m)\]

on \(\mathcal X\) where \(\Pi_i:=P[X_i]\), \(i \in \mathcal X\)

is called a stationary (or steady-state) distribution for the MC if it satisfies:

\[\sum_{i\in\mathcal X}\Pi_i p_{ij}\]\[\sum_{i\in\mathcal X}\Pi_i P[X_2=j|X_1=i]\]

or equivalently, in matrix form:

\[\Pi = \Pi Q\]

Note

\(\Pi\) is a left eigenvector of \(Q\) with eigenvalue 1.

Facts

  • For a finite-alphabet MC, \(\Pi\) always exists.

    • Also, if the MC is irreducible, then \(\Pi\) is unique

  • If a homogeneous MC is identically distributed, then it is stationary.

  • If a homogeneous MC, \(\{X_i\}\) has an initial state distribution given by \(\Pi\), i.e.

    \[P[X_1=i]=\Pi_i\]

    for every \(i \in\mathcal X\)

    then the MC is identically distributed and hence (by the above fact) it is stationary.
